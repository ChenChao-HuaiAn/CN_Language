## 检查清单

### 功能完整性

- [ ] 所有方法都已完整实现
- [ ] 无 TODO/FIXME/XXX 占位符
- [ ] 无空方法体
- [ ] 所有边界场景都已处理
- [ ] 所有异常情况都已捕获

### 参数校验

- [ ] 所有公共方法都有参数校验
- [ ] 空值检查完备
- [ ] 类型检查完备
- [ ] 边界值检查完备

### 异常处理

- [ ] 关键路径有 try-catch
- [ ] 异常信息清晰明确
- [ ] 有适当的日志记录
- [ ] 异常传播链路清晰

### 代码质量

- [ ] 无硬编码魔法值
- [ ] 变量/方法命名清晰
- [ ] 无重复代码
- [ ] 无注释掉的代码块

### 测试覆盖

- [ ] 单元测试已编写
- [ ] 核心逻辑已覆盖
- [ ] 边界条件已测试
- [ ] 异常场景已测试

### 规范一致性

- [ ] 代码风格符合项目规范
- [ ] 复杂逻辑有注释说明
- [ ] API 接口符合项目约定
- [ ] 正确使用了项目公共组件
