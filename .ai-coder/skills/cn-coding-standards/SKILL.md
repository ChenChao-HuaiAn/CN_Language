# CN语言编写规范

> **技能说明**：本技能用于指导AI编写符合CN语言语法规范的源代码。所有规范均来自 `plans/001 CN Language语法规范设计文档.md`。

---

## 何时使用此技能 / When to Use

**使用场景：**
- 用户要求编写CN语言代码（`.cn`文件）
- 用户要求创建或修改`.cn`文件
- 用户询问CN语言语法相关问题
- 用户要求实现CN语言功能模块
- 用户提到"CN语言"、"中文编程"等关键词

**不要使用：**
- 纯文档编写任务（不涉及CN代码）
- 项目规划或架构设计任务
- 其他编程语言的代码编写

### ✅ 类型系统规范

#### 基础类型
```cn
空类型    // 空类型
整数      // 整型（默认int32）
小数      // 浮点型（默认double）
布尔      // 布尔型
字符串    // 字符串型
```

#### 字符字面量
**语法**：字符字面量使用单引号 `'` 包围单个字符或转义序列。

```cn
变量 ch = 'A';      // 普通字符
变量 换行 = '\n';   // 换行符
变量 制表 = '\t';   // 制表符
变量 反斜杠 = '\\'; // 反斜杠
变量 单引号 = '\''; // 单引号
变量 空字符 = '\0'; // 空字符
变量 十六进制 = '\x41'; // 'A' 的十六进制表示
```

**支持的转义序列**：

| 转义序列 | 含义 | ASCII值 |
|---------|------|---------|
| `\n` | 换行符 | 10 |
| `\t` | 水平制表符 | 9 |
| `\r` | 回车符 | 13 |
| `\\` | 反斜杠 | 92 |
| `\'` | 单引号 | 39 |
| `\0` | 空字符 | 0 |
| `\xHH` | 十六进制表示 | HH |

**类型说明**：字符字面量的类型为 `整数`，值为对应字符的ASCII码。

**重要规则**：
- 字符字面量只能包含单个字符或转义序列
- 字符字面量与字符串字面量不同：`'A'` 是整数类型，`"A"` 是字符串类型

#### 指针类型
```cn
整数*     // 整数指针
整数**    // 二级指针
结构体名* // 结构体指针
字符*     // 字符指针（可变字符串）
```

#### 类型兼容性规则

CN语言支持以下隐式类型转换：

| 源类型 | 目标类型 | 允许 | 说明 |
|--------|---------|------|------|
| `字符` | `整数` | ✅ | 隐式转换 |
| `整数` | `字符` | ✅ | 截断低8位 |
| `字符[]` | `字符*` | ✅ | 数组退化 |
| `字符[]` | `字符串` | ✅ | 安全转换 |
| `字符*` | `字符串` | ✅ | 安全转换 |
| `字符串` | `字符*` | ✅ | 安全转换 |
| `字符串` | `字符[]` | ❌ | 禁止（不可变） |
| `空类型*` | 任意指针 | ✅ | 通用指针转换 |
| 任意指针 | `空类型*` | ✅ | 通用指针转换 |

**示例**：
```cn
字符* 可变字符串 = "Hello";
字符串 不可变字符串 = 可变字符串;  // ✅ 字符* -> 字符串

字符串 s = "World";
字符* p = s;  // ✅ 字符串 -> 字符*

空类型* 通用指针 = 分配内存(100);
整数* 整数指针 = 通用指针;  // ✅ 空类型* -> 整数*
```

#### 数组类型
```cn
整数 arr[10];       // 一维数组
整数 matrix[3][4];  // 多维数组
```

#### 结构体类型
```cn
结构体 点 {
    整数 x;
    整数 y;
}

// 结构体支持数组字段
结构体 数组字段测试 {
    整数[] 数组字段;        // 动态数组字段（指针）
    整数*[] 指针数组字段;   // 指针数组字段
    字符串[] 字符串数组;    // 字符串数组字段
    小数[10] 固定大小数组;  // 固定大小数组字段
}

// 结构体支持枚举类型字段
枚举 颜色 {
    红,
    绿,
    蓝
};

结构体 物体 {
    颜色 颜色值;  // 枚举类型字段
    整数 大小;
};
```

**数组字段语法**：
- `类型[] 字段名` - 动态数组字段，编译为指针类型
- `类型[N] 字段名` - 固定大小数组字段，编译为指针类型
- `类型*[] 字段名` - 指针数组字段，编译为二级指针

**枚举类型字段**：
- 结构体字段可以使用枚举类型
- 枚举类型字段支持赋值枚举成员
- 枚举类型字段支持 `==` 和 `!=` 比较操作

```cn
函数 主程序() -> 整数 {
    物体 球;
    球.颜色值 = 红;  // 枚举类型字段赋值
    
    如果 (球.颜色值 == 红) {  // 枚举类型字段比较
        返回 1;
    }
    
    返回 0;
}
```

#### 枚举类型
```cn
枚举 颜色 {
    红,        // 默认值 0
    绿,        // 默认值 1
    蓝 = 10,   // 显式赋值
    黄         // 值为 11
}

// 支持负数枚举值
枚举 状态 {
    成功 = 0,
    失败 = -1,      // 负数表示错误状态
    无效参数 = -2,
    超时 = -3
}

// 枚举类型作为结构体字段
结构体 任务 {
    字符串 名称;
    状态 当前状态;  // 枚举类型字段
};
```

### ✅ 函数定义规范

#### 基本语法
```
函数定义 ::= 重写修饰? 静态修饰? "函数" 标识符 "(" 参数列表? ")" ("->" 类型)? 代码块
函数原型声明 ::= 重写修饰? 静态修饰? "函数" 标识符 "(" 参数列表? ")" ("->" 类型)? ";"
```

#### 重要规则
1. **`重写`关键字位于函数声明开头**（不是末尾）
2. **返回类型使用 `-> 类型` 语法**（不是 `: 类型`）
3. 返回类型可省略，由编译器推断

#### 函数定义示例
```cn
// 基本函数
函数 加法(整数 a, 整数 b) -> 整数 {
    返回 a + b;
}

// 无返回值函数
函数 打印信息() {
    打印("你好");
}

// 重写函数（重写在开头）
重写 函数 发声() {
    打印("喵喵");
}

// 重写带返回类型
重写 函数 获取名称() -> 字符串 {
    返回 "猫";
}

// 重写静态函数
重写 静态 函数 工厂方法() -> 自身类型 {
    返回 创建实例();
}
```

#### 函数原型声明示例
**语法**：函数原型声明以分号结尾，无函数体。

```cn
// 前向声明（用于相互递归调用）
函数 计算阶乘(整数 n) -> 整数;

// 声明后定义
函数 主程序() -> 整数 {
    返回 计算阶乘(5);
}

函数 计算阶乘(整数 n) -> 整数 {
    如果 (n <= 1) {
        返回 1;
    }
    返回 n * 计算阶乘(n - 1);
}

// 类方法前向声明
类 动物 {
    虚拟 函数 发声();  // 抽象方法声明（无函数体）
}

// 静态方法声明
静态 函数 获取实例计数() -> 整数;
```

**函数原型声明的用途**：
1. **前向声明**：在函数定义前声明，支持相互递归调用
2. **接口定义**：在类中声明抽象方法（无函数体）
3. **API声明**：声明外部库函数，实现由链接器提供
4. **头文件模式**：声明与实现分离

**函数定义与函数原型声明的区别**：

| 特性 | 函数定义 | 函数原型声明 |
|------|---------|-------------|
| 函数体 | 有（代码块） | 无 |
| 结尾 | `}` | `;` |
| 用途 | 实现函数功能 | 声明函数签名 |
| 可否单独使用 | 可以 | 需要配套定义 |

#### 错误示例
```cn
// ❌ 错误：重写在末尾（旧语法）
函数 发声() 重写 {
    打印("喵喵");
}

// ❌ 错误：返回类型使用冒号（旧语法）
函数 获取名称(): 字符串 {
    返回 "猫";
}
```

#### 重写关键字限制
**重要**：`重写` 只能用于重写基类的 `虚拟` 函数，与C++语义一致。

```cn
// ✅ 正确：重写虚函数
类 动物 {
    虚拟 函数 发声() { 打印("动物"); }
}

类 狗 : 动物 {
    重写 函数 发声() { 打印("汪汪"); }  // 正确：基类是虚函数
}

// ❌ 错误：重写非虚函数
类 基类 {
    函数 普通方法() { 打印("基类"); }  // 非虚函数
}

类 子类 : 基类 {
    重写 函数 普通方法() { }  // 错误！基类方法不是虚函数
}
```

**设计原因**：
- 虚函数 + 重写 = 运行时多态（动态绑定）
- 非虚函数 + 同名函数 = 隐藏（静态绑定），不能使用 `重写` 关键字

### ✅ 模块系统规范

#### 文件即模块
CN语言采用"文件即模块"设计，每个源文件（`.cn`）自动成为一个独立的模块。

**模块命名规则**：
- 模块名由文件名决定，例如 `数学工具.cn` 的模块名为 `数学工具`
- 文件名支持中文、英文和下划线
- 模块名与文件路径相关，支持相对路径和绝对路径导入

**项目目录结构示例**：
```
项目目录结构：
├── 主程序.cn          # 模块名: 主程序
├── 工具/
│   ├── 数学工具.cn    # 模块名: 工具/数学工具
│   └── 字符串工具.cn  # 模块名: 工具/字符串工具
└── 库/
    └── 网络库.cn      # 模块名: 库/网络库
```

#### 可见性控制
```cn
// 文件级可见性（在文件顶部声明）
公开:    // 以下声明为公开，可被其他模块导入
函数 外部接口() { }

私有:    // 以下声明为私有，仅本文件可见
函数 内部实现() { }
```

#### 导入语句
```cn
// 全量导入
导入 数学工具;

// 选择性导入
导入 数学工具 { 加法, 减法 };

// 别名导入
导入 数学工具(数学);

// 路径导入
导入 ./兄弟模块;
导入 ../父级模块;
导入 工具/数学/高级;
```

#### 从...导入
```cn
从 数学工具 导入 { 加法, 减法 };
从 工具.数学 导入 *;
从 ./兄弟模块 导入 { 功能A };
```

### ✅ 异常处理规范

```cn
// 基本try-catch
尝试 {
    可能抛出异常的代码();
} 捕获 (运行时异常 e) {
    打印("捕获到异常: " + e.消息);
}

// try-catch-finally
尝试 {
    打开文件("data.txt");
} 捕获 (输入输出异常 e) {
    打印("IO错误: " + e.消息);
} 最终 {
    关闭文件();
}

// 抛出异常
抛出 "运行时异常" "发生错误";
```

```cn
// 导入语句
```

### 类定义模板
```cn
类 示例类 {
公开:
    // 公开字段
    整数 公开字段;
    字符串 名称;
    
    // 默认构造函数
    函数 示例类() {
        自身.公开字段 = 0;

    }
    
    // 带参数构造函数 + 初始化列表
    函数 示例类(整数 值, 字符串 名) : 公开字段(值), 名称(名) {

    // 公开方法

        返回 自身.公开字段;
    }
    
私有:
    // 私有字段
    整数 私有字段;
    
保护:
    // 保护字段
    整数 保护字段;
}
```

## 常见错误

### 1. 重写关键字位置错误
```cn
// ❌ 错误
函数 方法() 重写 { }

// ✅ 正确
重写 函数 方法() { }
```

### 2. 返回类型语法错误
```cn
// ❌ 错误
函数 方法(): 整数 { }

// ✅ 正确
函数 方法() -> 整数 { }
```

### 3. 静态变量类型推断
```cn
// ❌ 错误：静态变量不支持类型推断
静态 计数 = 0;

// ✅ 正确：必须显式指定类型
静态 整数 计数 = 0;
```

### 4. 使用预留关键字
```cn
// ❌ 错误：命名空间是预留关键字
命名空间 我的命名空间 { }
```

### 5. 混淆字符字面量与字符串字面量
```cn
// ❌ 错误：字符字面量是整数类型，不能赋值给字符串
字符串 s = 'A';

// ✅ 正确：使用字符串字面量
字符串 s = "A";

// ✅ 正确：字符字面量赋值给整数
整数 ch = 'A';
```
