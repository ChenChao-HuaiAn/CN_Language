## 系统要求

### 支持的操作系统

| 系统 | 版本 | 状态 |
|------|------|------|
| Windows | 10/11 | ✅ 完全支持 |
| Linux | Ubuntu 20.04+ | ✅ 完全支持 |
| macOS | 11.0+ | ✅ 完全支持 |

### 必需工具

| 工具 | 最低版本 | 用途 |
|------|----------|------|
| CMake | 3.15 | 构建系统 |
| GCC/Clang | 9.0 | C 编译器 |
| Git | 2.0 | 版本控制 |

### 可选工具

| 工具 | 用途 |
|------|------|
| QEMU | 内核测试 |
| GDB/LLDB | 调试 |
| Valgrind | 内存检查 |
| gcov | 覆盖率分析 |

## 目录结构

```
CN_Language/
├── src/                    # 源代码
│   ├── frontend/          # 前端编译器
│   ├── ir/                # 中间表示
│   ├── backend/           # 后端代码生成
│   ├── semantics/         # 语义分析
│   ├── runtime/           # 运行时库
│   └── cli/               # 命令行工具
├── tests/                  # 测试
│   ├── unit/              # 单元测试
│   ├── integration/       # 集成测试
│   └── system/            # 系统测试
├── tools/                  # 工具
│   └── vscode/            # VSCode 扩展
├── examples/               # 示例代码
├── plans/                  # 设计文档
└── docs/                   # 文档
```
