# 词法分析器 (Lexer)

## 概述

词法分析器是编译器的第一个阶段，负责将源代码字符流转换为标记（Token）流。CN_Language 的词法分析器完全支持中文标识符和关键字。

[`src/frontend/lexer/lexer.c`](../../src/frontend/lexer/lexer.c:1)

## 核心数据结构

### CnLexer 结构

```c
typedef struct CnLexer {
    const char *source;      // 源代码字符串
    size_t length;           // 源代码长度
    size_t offset;           // 当前扫描位置
    size_t line;             // 当前行号
    size_t column;           // 当前列号
    const char *filename;    // 源文件名
    CnDiagnostics *diagnostics;  // 诊断信息
} CnLexer;
```

### CnToken 结构

```c
typedef struct CnToken {
    CnTokenKind kind;        // 标记类型
    const char *start;       // 标记起始位置
    size_t length;           // 标记长度
    size_t line;             // 行号
    size_t column;           // 列号
    union {
        long int_value;      // 整数值
        double float_value;  // 浮点数值
        // 字符串值通过 start 和 length 获取
    } value;
} CnToken;
```

## 核心函数

### 初始化词法分析器

```c
void cn_frontend_lexer_init(CnLexer *lexer, const char *source, size_t length, const char *filename);
```

**参数：**
- `lexer`: 词法分析器实例
- `source`: 源代码字符串
- `length`: 源代码长度
- `filename`: 源文件名（用于错误报告）

**示例：**
```c
CnLexer lexer;
const char *source = "函数 主程序 () { 返回 0; }";
cn_frontend_lexer_init(&lexer, source, strlen(source), "test.cn");
```

### 获取下一个标记

```c
int cn_frontend_lexer_next_token(CnLexer *lexer, CnToken *token);
```

**参数：**
- `lexer`: 词法分析器实例
- `token`: 输出标记

**返回值：**
- `1`: 成功获取标记
- `0`: 到达文件末尾

**示例：**
```c
CnToken token;
while (cn_frontend_lexer_next_token(&lexer, &token)) {
    // 处理标记
}
```

## 扫描逻辑

### 标识符识别

```c
static int is_identifier_start(unsigned char c)
{
    // 字母或下划线开头
    if (c == '_' || isalpha(c)) {
        return 1;
    }
    // 支持 UTF-8 中文（>= 0x80）
    if (c >= 0x80) {
        return 1;
    }
    return 0;
}

static int is_identifier_continue(unsigned char c)
{
    // 字母、数字或下划线
    if (isalnum(c) || c == '_') {
        return 1;
    }
    // 支持 UTF-8 中文
    if (c >= 0x80) {
        return 1;
    }
    return 0;
}
```

### 数字字面量

支持以下格式：
- 十进制整数：`123`
- 十六进制整数：`0xFF`
- 八进制整数：`0755`
- 二进制整数：`0b1010`
- 浮点数：`3.14`
- 科学计数法：`1.5e10`、`2.5E-3`
- 类型后缀：`100L`、`3.14f`、`1000ULL`

### 字符串字面量

```c
// 处理转义序列
static char* process_string_escapes(const char *raw_string, size_t raw_length, size_t *out_length)
```

## 错误处理

词法分析器通过诊断系统报告错误：

```c
static void report_lex_error(CnLexer *lexer, CnDiagCode code, const char *message)
{
    if (!lexer || !lexer->diagnostics) {
        return;
    }

    cn_support_diagnostics_report(lexer->diagnostics,
                                  CN_DIAG_SEVERITY_ERROR,
                                  code,
                                  lexer->filename,
                                  lexer->line,
                                  lexer->column,
                                  message);
}
```

### 常见错误

| 错误代码 | 描述 | 示例 |
|----------|------|------|
| `CN_DIAG_CODE_LEX_INVALID_CHAR` | 无效字符 | 包含非法符号 |
| `CN_DIAG_CODE_LEX_UNTERMINATED_STRING` | 未终止的字符串 | `"hello` |
| `CN_DIAG_CODE_LEX_INVALID_NUMBER` | 无效数字 | `123abc` |
| `CN_DIAG_CODE_LEX_INVALID_ESCAPE` | 无效转义序列 | `"\z"` |

## 使用示例

```c
#include "cnlang/frontend/lexer.h"
#include "cnlang/support/diagnostics.h"
#include <stdio.h>
#include <string.h>

int main() {
    const char *source = 
        "函数 主程序 () {\n"
        "    打印 (\"你好，世界！\");\n"
        "    返回 0;\n"
        "}";
    
    CnLexer lexer;
    CnDiagnostics diagnostics;
    
    // 初始化诊断系统
    cn_support_diagnostics_init(&diagnostics);
    
    // 初始化词法分析器
    cn_frontend_lexer_init(&lexer, source, strlen(source), "test.cn");
    lexer.diagnostics = &diagnostics;
    
    // 扫描所有标记
    CnToken token;
    while (cn_frontend_lexer_next_token(&lexer, &token)) {
        printf("Token: kind=%d, line=%zu, column=%zu\n", 
               token.kind, token.line, token.column);
    }
    
    return 0;
}
```

## 性能优化

1. **单次扫描**：词法分析器只扫描源代码一次，时间复杂度 O(n)
2. **零拷贝**：标记直接引用源代码字符串，避免字符串复制
3. **快速关键字查找**：使用哈希表或完美哈希进行关键字匹配
