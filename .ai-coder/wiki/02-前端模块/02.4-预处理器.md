# 预处理器 (Preprocessor)

## 概述

预处理器在词法分析之前处理源代码，执行宏展开、条件编译和文件包含等操作。CN_Language 的预处理器采用 C 风格的预处理语法，同时支持中文标识符。

## 文件位置

[`src/frontend/preprocessor/preprocessor.c`](../../src/frontend/preprocessor/preprocessor.c:1)

## 核心功能

### 宏定义

#### 对象宏

```c
// 语法：#定义 宏名 替换文本
#定义 最大 100
#定义 真 1
#定义 假 0
#定义 版本 "1.0.0"
```

#### 函数宏

```c
// 语法：#定义 宏名 (参数列表) 替换文本
#定义 平方 (x) ((x) * (x))
#定义 最小 (a, b) ((a) < (b) ? (a) : (b))
#定义 最大 (a, b) ((a) > (b) ? (a) : (b))
#定义 绝对值 (x) ((x) < 0 ? -(x) : (x))
```

#### 宏展开示例

```cn
#定义 半径 5
#定义 圆面积 (r) (3.14 * (r) * (r))

函数 主程序 () {
    整数 面积 = 圆面积 (半径);
    // 展开后：
    // 整数 面积 = (3.14 * (5) * (5));
}
```

### 条件编译

#### 基本语法

```c
// #如果 - #否则 - #结束
#如果 定义 调试模式
    打印 ("调试模式已启用");
#否则
    打印 ("发布模式");
#结束

// #如果 - #否则如果 - #否则 - #结束
#如果 系统 == "Windows"
    #定义 路径分隔符 "\\"
#否则如果 系统 == "Linux"
    #定义 路径分隔符 "/"
#否则如果 系统 == "MacOS"
    #定义 路径分隔符 "/"
#否则
    #定义 路径分隔符 "/"
#结束
```

#### 条件指令

| 指令 | 描述 |
|------|------|
| `#如果` | 如果条件为真 |
| `#否则如果` | 否则如果 |
| `#否则` | 否则 |
| `#结束` | 结束条件块 |
| `#定义` | 定义宏 |
| `#未定义` | 取消定义宏 |
| `#包含` | 包含文件 |

#### 条件表达式

```c
// 检查宏是否定义
#如果 定义 调试模式
#如果 未定义 发布模式

// 比较表达式
#如果 版本 > 1
#如果 系统 == "Windows"

// 逻辑运算
#如果 定义 调试模式 && 版本 > 1
#如果 定义 测试模式 || 定义 调试模式
```

### 文件包含

```c
// 语法：#包含 "文件名" 或 #包含 <文件名>
#包含 "头文件.cn"
#包含 "模块/子模块.cn"
#包含 <标准库/输入输出>

// 条件包含
#如果 定义 使用扩展
    #包含 "扩展功能.cn"
#结束
```

### 特殊宏

```c
// 行号
#行号  // 展开为当前行号

// 文件名
#文件  // 展开为当前文件名

// 日期和时间
#日期  // 展开为当前日期
#时间  // 展开为当前时间

// 使用示例
函数 调试信息 () {
    打印 ("文件：%s, 行号：%d\n", #文件，#行号);
}
```

### 指令列表

| 指令 | 描述 | 示例 |
|------|------|------|
| `#定义` | 定义宏 | `#定义 最大 100` |
| `#未定义` | 取消定义宏 | `#未定义 调试模式` |
| `#包含` | 包含文件 | `#包含 "头文件.cn"` |
| `#如果` | 条件开始 | `#如果 定义 调试模式` |

### 宏参数

```c
// 带参数的宏
#定义 加法 (a, b) ((a) + (b))
#定义 乘法 (a, b) ((a) * (b))

// 可变参数宏
#定义 日志 (格式，...) 打印 ("[日志] " 格式，##__VA_ARGS__)
#定义 调试 (格式，...) 打印 ("[调试] %s:%d: " 格式，#文件，#行号，##__VA_ARGS__)

// 使用
日志 ("值=%d", x);
调试 ("处理中...");
```

### 字符串化运算符

```c
// # 运算符将宏参数转换为字符串
#定义 字符串化 (x) #x
#定义 打印变量 (var) 打印 (#var " = %d\n", var)

// 使用
整数 x = 10;
打印变量 (x);  // 展开为：打印 ("x = %d\n", x);
```

### 连接运算符

```c
// ## 运算符连接两个标记
#定义 连接 (a, b) a##b
#定义 创建函数名 (前缀，后缀) 前缀##_##后缀

// 使用
函数 连接 (加法，_impl) (整数 a, 整数 b) {  // 展开为：函数 加法_impl(...)
    返回 a + b;
}
```

## 实现细节

### 预处理器状态机

```c
typedef struct CnPreprocessor {
    const char *source;          // 源代码
    size_t length;               // 源代码长度
    size_t position;             // 当前位置
    size_t line;                 // 当前行号
    size_t column;               // 当前列号
    const char *filename;        // 文件名
    
    CnHashMap *macros;           // 宏定义表
    CnArray *include_stack;      // 包含文件栈
    CnArray *output;             // 输出缓冲区
    
    CnDiagnostics *diagnostics;  // 诊断信息
} CnPreprocessor;
```

### 宏展开算法

```c
// 宏展开伪代码
string expand_macros(string input) {
    output = ""
    while input not empty {
        token = get_next_token(input)
        if token is macro_name {
            if macro has arguments {
                args = parse_arguments(input)
                output += expand_macro_with_args(macro, args)
            } else {
                output += expand_object_macro(macro)
            }
        } else {
            output += token
        }
    }
    return output
}
```

### 条件编译处理

```c
// 条件编译处理
void process_conditional(CnPreprocessor *pp) {
    int condition = evaluate_condition(pp);
    
    if (condition) {
        // 处理真分支
        process_block(pp);
    } else {
        // 跳过假分支
        skip_block(pp);
    }
}
```

## 错误处理

### 常见错误

| 错误 | 描述 | 示例 |
|------|------|------|
| 宏重定义 | 同一宏被多次定义 | `#定义 X 1` 然后 `#定义 X 2` |
| 未终止的条件 | 缺少 `#结束` | `#如果 定义 X` 没有对应的 `#结束` |
| 文件未找到 | 包含的文件不存在 | `#包含 "不存在的文件.cn"` |
| 无效的参数 | 宏参数数量不匹配 | `#定义 F(a,b) ...` 调用 `F(1)` |
| 嵌套过深 | 包含嵌套层数过多 | 循环包含 |

### 错误报告

```c
static void report_pp_error(CnPreprocessor *pp, const char *message)
{
    if (!pp || !pp->diagnostics) {
        return;
    }

    cn_support_diagnostics_report(pp->diagnostics,
                                  CN_DIAG_SEVERITY_ERROR,
                                  CN_DIAG_CODE_PP_ERROR,
                                  pp->filename,
                                  pp->line,
                                  pp->column,
                                  message);
}
```

## 使用示例

### 完整的预处理器示例

```cn
// 配置文件 config.cn
#定义 调试模式 1
#定义 版本 1
#定义 最大连接数 100

// 主程序
#包含 "config.cn"

#如果 定义 调试模式
    #定义 日志 (msg) 打印 ("[调试] %s\n", msg)
#否则
    #定义 日志 (msg)  // 空宏
#结束

函数 主程序 () {
    日志 ("程序启动");
    
    整数 连接数 = 0;
    当 (连接数 < 最大连接数) {
        // 处理连接
        连接数 = 连接数 + 1;
    }
    
    #如果 版本 > 1
        打印 ("新版本功能");
    #结束
    
    返回 0;
}
```

```cn
    #定义 路径分隔符 "\\"

#否则如果 系统 == "Linux"
    #定义 换行符 "\n"
    #定义 路径分隔符 "/"
    #包含 <unistd.h>
#否则如果 系统 == "MacOS"
    #定义 换行符 "\n"
    #定义 路径分隔符 "/"
    #包含 <unistd.h>
#否则
    #错误 "不支持的操作系统"
#结束

函数 打印换行 () {
    打印 (换行符);
}
```

## 性能优化

1. **宏缓存**：缓存已展开的宏，避免重复展开
2. **条件编译短路**：在条件为假时快速跳过代码块
3. **包含文件缓存**：缓存已包含的文件内容，避免重复读取
