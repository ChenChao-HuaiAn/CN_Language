# 前端模块文档

前端模块是 CN_Language 编译器的核心部分，负责将源代码转换为抽象语法树（AST）。前端模块包含以下子模块：

## 模块结构

```
src/frontend/
├── lexer/           # 词法分析器
│   ├── lexer.c     # 词法分析主逻辑
│   ├── token.c     # 标记定义
│   └── keywords.c  # 关键字表
├── parser/          # 语法分析器
│   └── parser.c    # 语法分析主逻辑
├── ast/             # 抽象语法树
│   └── ast.c       # AST 节点定义和操作
├── preprocessor/    # 预处理器
│   └── preprocessor.c
└── module_loader/   # 模块加载器
    └── module_loader.c
```

## 编译流程

```
源代码 (.cn) 
    ↓
[预处理器] → 展开宏、处理条件编译
    ↓
[词法分析器] → 生成标记流
    ↓
[语法分析器] → 构建抽象语法树
    ↓
[语义分析器] → 类型检查、符号解析
    ↓
[中间表示] → 生成 IR
```
