# 语法分析器 (Parser)

## 概述

语法分析器是编译器前端的核心组件，负责将词法分析器生成的标记流转换为抽象语法树（AST）。CN_Language 的语法分析器采用递归下降算法实现，完全支持中文语法结构。

## 文件位置

[`src/frontend/parser/parser.c`](../../src/frontend/parser/parser.c:1)

## 核心数据结构

### CnParser 结构

```c
typedef struct CnParser {
    CnLexer *lexer;                  // 词法分析器
    CnToken current;                 // 当前标记
    int has_current;                 // 是否有当前标记
    int error_count;                 // 错误计数
    CnDiagnostics *diagnostics;      // 诊断信息
    CnVisibility current_visibility; // 当前可见性
} CnParser;
```

## 语法结构

### 程序结构

```
Program
    ├── ModuleDecl?           # 模块声明（可选）
    ├── ImportStmt*           # 导入语句（零或多个）
    ├── (FunctionDecl | 
    │     StructDecl | 
    │     EnumDecl | 
    │     GlobalVarDecl)*     # 顶级声明
    └── EOF
```

### 表达式优先级

从低到高：

| 优先级 | 运算符 | 结合性 |
|--------|--------|--------|
| 1 | `=`, `+=`, `-=`, `*=`, `/=` | 右结合 |
| 2 | `?:` (三元运算符) | 右结合 |
| 3 | `||` | 左结合 |
| 4 | `&&` | 左结合 |
| 5 | `|` | 左结合 |
| 6 | `^` | 左结合 |
| 7 | `&` | 左结合 |
| 8 | `==`, `!=` | 左结合 |
| 9 | `<`, `>`, `<=`, `>=` | 左结合 |
| 10 | `<<`, `>>` | 左结合 |
| 11 | `+`, `-` | 左结合 |
| 12 | `*`, `/`, `%` | 左结合 |
| 13 | `!`, `~`, `-` (一元) | 右结合 |
| 14 | `()`, `[]`, `.`, `->` | 左结合 |

## 核心函数

### 创建和销毁

```c
CnParser *cn_frontend_parser_new(CnLexer *lexer);
void cn_frontend_parser_free(CnParser *parser);
```

### 设置诊断

```c
void cn_frontend_parser_set_diagnostics(CnParser *parser, CnDiagnostics *diagnostics);
```

### 解析程序

```c
bool cn_frontend_parse_program(CnParser *parser, CnAstProgram **out_program);
```

**参数：**
- `parser`: 语法分析器实例
- `out_program`: 输出的 AST 程序节点

**返回值：**
- `true`: 解析成功（无错误）
- `false`: 解析失败（有错误）

**示例：**
```c
CnLexer lexer;
CnParser *parser = cn_frontend_parser_new(&lexer);
CnAstProgram *program;

if (cn_frontend_parse_program(parser, &program)) {
    // 解析成功，处理 AST
} else {
    // 解析失败，处理错误
}
```

## 语句解析

### 函数声明

```c
// 语法：
// 函数 函数名 (参数列表) [中断处理程序 (中断号)] { 函数体 }

函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

// 中断处理程序
中断处理程序 定时器处理 (整数 中断号) {
    // 处理定时器中断
}
```

### 结构体声明

```c
// 语法：
// 结构体 结构体名 { 字段列表 }

结构体 学生 {
    字符串 姓名;
    整数 年龄;
    小数 成绩;
}
```

### 枚举声明

```c
// 语法：
// 枚举 枚举名 { 成员列表 }

枚举 颜色 {
    红，
    绿，
    蓝
}
```

### 模块声明

```c
// 语法：
// 模块 模块名;

模块 数学模块;

导出 函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}
```

### 导入语句

```c
// 完整导入
导入 数学模块;

// 选择性导入
从 数学模块 导入 加法，减法;

// 带别名导入
从 数学模块 导入 乘法 作为 乘;
```

### 变量声明

```c
// 语法：
// 变量 类型 名称 = 初始值;
// 类型 名称 = 初始值;  // 简写

变量 整数 x = 10;
整数 y = 20;  // 简写

// 全局变量
整数 全局计数 = 0;
```

### 控制流语句

#### 如果 - 否则

```c
// 语法：
// 如果 (条件) { 语句块 }
// 如果 (条件) { 语句块 } 否则 { 语句块 }
// 如果 (条件) { 语句块 } 否则 如果 (条件) { 语句块 } 否则 { 语句块 }

如果 (分数 >= 90) {
    打印 ("优秀");
} 否则 如果 (分数 >= 60) {
    打印 ("及格");
} 否则 {
    打印 ("不及格");
}
```

#### 当循环

```c
// 语法：
// 当 (条件) { 语句块 }

整数 i = 0;
当 (i < 10) {
    打印 (i);
    i = i + 1;
}
```

#### 循环语句 (for)

```c
// 语法：
// 循环 (初始化; 条件; 更新) { 语句块 }

循环 (整数 i = 0; i < 10; i = i + 1) {
    打印 (i);
}
```

#### 选择语句 (switch)

```c
// 语法：
// 选择 (表达式) { 情况 值：语句块; 默认：语句块; }

整数 选项 = 2;
选择 (选项) {
    情况 1:
        打印 ("选项一");
        中断;
    情况 2:
        打印 ("选项二");
        中断;
    默认:
        打印 ("未知选项");
}
```

#### 中断和继续

```c
循环 (整数 i = 0; i < 100; i = i + 1) {
    如果 (i == 50) {
        中断;  // 跳出循环
    }
    如果 (i % 2 == 0) {
        继续;  // 跳过本次迭代
    }
    打印 (i);
}
```

## 错误恢复

语法分析器使用以下策略进行错误恢复：

### 同步标记

```c
// 在遇到错误时，跳过直到找到同步标记
static void synchronize(CnParser *parser)
{
    parser_advance(parser);
    
    while (parser->current.kind != CN_TOKEN_EOF) {
        // 同步标记：语句结束符或声明开始
        if (parser->current.kind == CN_TOKEN_SEMICOLON ||
            parser->current.kind == CN_TOKEN_KEYWORD_FUNC ||
            parser->current.kind == CN_TOKEN_KEYWORD_STRUCT ||
            parser->current.kind == CN_TOKEN_KEYWORD_ENUM) {
            return;
        }
        
        parser_advance(parser);
    }
}
```

### 错误报告

```c
static int parser_expect(CnParser *parser, CnTokenKind kind)
{
    if (parser_match(parser, kind)) {
        return 1;
    }

    parser->error_count++;
    if (parser->diagnostics) {
        cn_support_diagnostics_report(parser->diagnostics,
                                      CN_DIAG_SEVERITY_ERROR,
                                      CN_DIAG_CODE_PARSE_EXPECTED_TOKEN,
                                      parser->lexer->filename,
                                      parser->current.line,
                                      parser->current.column,
                                      "语法错误：缺少预期的标记");
    }
    return 0;
}
```

## 使用示例

```c
#include "cnlang/frontend/parser.h"
#include "cnlang/frontend/lexer.h"
#include "cnlang/support/diagnostics.h"
#include <stdio.h>
#include <string.h>

int main() {
    const char *source = 
        "函数 主程序 () {\n"
        "    打印 (\"你好，世界！\");\n"
        "    返回 0;\n"
        "}";
    
    CnLexer lexer;
    CnParser *parser;
    CnDiagnostics diagnostics;
    CnAstProgram *program;
    
    // 初始化
    cn_frontend_lexer_init(&lexer, source, strlen(source), "test.cn");
    cn_support_diagnostics_init(&diagnostics);
    
    parser = cn_frontend_parser_new(&lexer);
    cn_frontend_parser_set_diagnostics(parser, &diagnostics);
    
    // 解析
    if (cn_frontend_parse_program(parser, &program)) {
        printf("解析成功！\n");
        // 处理 AST...
    } else {
        printf("解析失败，发现 %d 个错误\n", parser->error_count);
    }
    
    // 清理
    cn_frontend_parser_free(parser);
    
    return 0;
}
```
