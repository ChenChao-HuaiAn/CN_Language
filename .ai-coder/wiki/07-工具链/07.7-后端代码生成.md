# 后端代码生成

## 概述

后端代码生成将优化后的 IR 转换为目标代码。CN_Language 目前支持生成 C 代码，通过 C 编译器生成机器码。

## 文件位置

[`src/backend/cgen/cgen.c`](../../src/backend/cgen/cgen.c:1)
[`src/backend/cgen/module_cgen.c`](../../src/backend/cgen/module_cgen.c:1)

## 代码生成流程

```
IR → [C 代码生成] → C 源代码 → [C 编译器] → 目标代码
```

## IR 到 C 的转换

### 函数生成

```c
void cgen_function(CnCodeGen *codegen, CnIrFunction *func) {
    // 生成函数签名
    emit_type(codegen, func->return_type);
    emit(" %s(", c_name(func->name));
    
    // 生成参数
    for (size_t i = 0; i < func->param_count; i++) {
        if (i > 0) emit(", ");
        emit_operand(codegen, func->params[i]);
    }
    emit(") {\n");
    
    // 生成局部变量声明
    indent(codegen);
    for (size_t i = 0; i < func->local_count; i++) {
        emit_type(codegen, func->locals[i].type);
        emit(" %s;\n", c_name(func->locals[i].name));
    }
    emit("\n");
    
    // 生成基本块
    CnIrBasicBlock *block = func->first_block;
    while (block) {
        cgen_block(codegen, block);
        block = block->next;
    }
    
    emit("}\n\n");
}
```

### 指令生成

```c
void cgen_inst(CnCodeGen *codegen, CnIrInst *inst) {
    indent(codegen);
    
    switch (inst->kind) {
        case CN_IR_INST_MOV:
            cgen_mov(codegen, inst);
            break;
        case CN_IR_INST_ADD:
            cgen_binary(codegen, inst, "+");
            break;
        case CN_IR_INST_SUB:
            cgen_binary(codegen, inst, "-");
            break;
        case CN_IR_INST_MUL:
            cgen_binary(codegen, inst, "*");
            break;
        case CN_IR_INST_DIV:
            cgen_binary(codegen, inst, "/");
            break;
        case CN_IR_INST_LOAD:
            cgen_load(codegen, inst);
            break;
        case CN_IR_INST_STORE:
            cgen_store(codegen, inst);
            break;
        case CN_IR_INST_CALL:
            cgen_call(codegen, inst);
            break;
        case CN_IR_INST_RET:
            cgen_ret(codegen, inst);
            break;
        case CN_IR_INST_JMP:
            cgen_jump(codegen, inst);
            break;
        // ... 其他指令
    }
}
```

### 二元运算

```c
void cgen_binary(CnCodeGen *codegen, CnIrInst *inst, const char *op) {
    emit("%s = %s %s %s;\n",
         reg_name(inst->dest),
         operand_to_c(inst->src1),
         op,
         operand_to_c(inst->src2));
}

// 示例:
// IR:  %3 = ADD %1, %2
// C:   int _3 = _1 + _2;
```

### 函数调用

```c
void cgen_call(CnCodeGen *codegen, CnIrInst *inst) {
    emit("%s = ", reg_name(inst->dest));
    emit("%s(", func_name(inst->src1));
    
    for (size_t i = 0; i < inst->extra_args_count; i++) {
        if (i > 0) emit(", ");
        emit("%s", operand_to_c(inst->extra_args[i]));
    }
    
    emit(");\n");
}

// 示例:
// IR:  %2 = CALL @加法，[%0, %1]
// C:   int _2 = 加法 (_0, _1);
```

### 条件跳转

```c
void cgen_cond_jump(CnCodeGen *codegen, CnIrInst *inst) {
    // JZ %cond, %label
    // 转换为: if (!cond) goto label;
    emit("if (!%s) goto %s;\n",
         operand_to_c(inst->src1),
         block_label(inst->src2));
}

// 示例:
// IR:  JZ %cond, loop.exit
// C:   if (!cond) goto loop_exit;
```

### 内存操作

```c
void cgen_load(CnCodeGen *codegen, CnIrInst *inst) {
    // LOAD %src
    // 转换为: *src
    emit("%s = *%s;\n",
         reg_name(inst->dest),
         operand_to_c(inst->src1));
}

void cgen_store(CnCodeGen *codegen, CnIrInst *inst) {
    // STORE %value, %dest
    // 转换为: *dest = value
    emit("*%s = %s;\n",
         operand_to_c(inst->src2),
         operand_to_c(inst->src1));
}
```

## 类型转换

### 基础类型映射

| CN_Language | C |
|-------------|---|
| `整数` | `long long` |
| `小数` | `double` |
| `布尔` | `int` |
| `字符` | `char` |

## 模块代码生成

```c
void cgen_module(CnCodeGen *codegen, CnIrModule *module) {
    // 生成头文件包含
    emit("#include <stdio.h>\n");
    emit("#include <stdlib.h>\n");
    emit("#include \"cn_runtime.h\"\n\n");
    
    // 前向声明
    for (CnIrFunction *func = module->first_func; func; func = func->next) {
        emit_function_decl(codegen, func);
    }
    emit("\n");
    
    // 生成全局变量
    for (CnIrGlobalVar *global = module->first_global; global; global = global->next) {
        emit_global(codegen, global);
    }
    emit("\n");
    
    // 生成函数
    for (CnIrFunction *func = module->first_func; func; func = func->next) {
        cgen_function(codegen, func);
    }
}
```

int cn_cgen_emit(CnCodeGen *codegen, const char *filename) {
    FILE *f = fopen(filename, "w");
    if (!f) return -1;
    
    const char *code = string_builder_cstr(codegen->output);
    fprintf(f, "%s", code);
    
    fclose(f);
    return 0;
}

## 编译和链接

### 使用 GCC 编译

```bash
# 生成 C 代码
cnc 程序.cn --emit-c -o 程序.c

# 编译为可执行文件
gcc 程序.c -lcn_runtime -o 程序
```

### 使用 MSVC 编译

```bash
# 生成 C 代码
cnc 程序.cn --emit-c -o 程序.c

# 编译为可执行文件
cl 程序.c cn_runtime.lib /Fe:程序.exe
```
