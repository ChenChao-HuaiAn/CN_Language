# 格式化工具 (cnfmt)

## 概述

`cnfmt` 是 CN_Language 的代码格式化工具，自动格式化代码以保持一致的风格。

## 文件位置

[`src/format/formatter.c`](../../src/format/formatter.c:1)
[`src/cli/cnfmt/main.c`](../../src/cli/cnfmt/main.c:1)

## 使用方法

### 基本用法

```bash
# 格式化文件
cnfmt 文件.cn

# 格式化多个文件
cnfmt 文件 1.cn 文件 2.cn

# 检查格式（不修改）
cnfmt --check 文件.cn

# 输出到标准输出
cnfmt --stdout 文件.cn

# 递归格式化目录
cnfmt -r 目录/
```

### 命令行选项

| 选项 | 描述 |
|------|------|
| `--check` | 检查格式，不修改文件 |
| `--stdout` | 输出到标准输出 |
| `-r`, `--recursive` | 递归处理目录 |
| `--style <风格>` | 格式化风格 |
| `--tab-width <n>` | 制表符宽度 |
| `--indent-width <n>` | 缩进宽度 |
| `-h`, `--help` | 显示帮助 |

## 格式化规则

### 缩进

```cn
// 使用 4 个空格缩进
函数 主程序 () {
    如果 (条件) {
        打印 ("缩进 4 空格");
    }
}
```

### 空格

```cn
// 运算符两侧添加空格
整数 结果 = a + b;

// 逗号后添加空格
函数 调用 (参数 1, 参数 2, 参数 3);

// 关键字后添加空格
如果 (条件) { }
当 (条件) { }
循环 (初始化; 条件; 更新) { }

// 函数名后不添加空格
函数 主程序 () { }
```

### 空行

```cn
// 函数之间空一行
函数 函数 1() {
    // ...
}

函数 函数 2() {
    // ...
}

// 逻辑块之间空一行
函数 处理 () {
    // 初始化
    整数 x = 0;
    
    // 处理
    当 (x < 10) {
        x = x + 1;
    }
    
    // 清理
    释放资源 ();
}
```

### 行长度

```cn
// 最大行长度 100 字符
字符串 长消息 = "这是一条非常长的消息，超过了推荐的行长度，应该被分割成多行";

// 长表达式分行
整数 结果 = 非常长的函数名 (
    参数 1,
    参数 2,
    参数 3
);
```

### 括号

```cn
// 左括号在行尾
函数 主程序 () {
    如果 (条件) {
        // ...
    }
}

// 右括号单独一行
函数 主程序 () {
    如果 (条件) {
        // ...
    } 否则 {
        // ...
    }
}
```

## 配置选项

### 配置文件

```json
// .cnfmt.json
{
    "tabWidth": 4,
    "indentWidth": 4,
    "maxLineLength": 100,
    "useTabs": false,
    "insertFinalNewline": true,
    "trimTrailingWhitespace": true
}
```

### 风格预设

```bash
# 默认风格
cnfmt --style default 文件.cn

# 紧凑风格
cnfmt --style compact 文件.cn

# 宽松风格
cnfmt --style wide 文件.cn
```

## 格式化示例

### 格式化前

```cn
函数 主程序(){如果 (真){打印 ("你好");}返回 0;}函数 其他 (){// 注释 整数 x=10;}
```

### 格式化后

```cn
函数 主程序 () {
    如果 (真) {
        打印 ("你好");
    }
    返回 0;
}

函数 其他 () {
    // 注释
    整数 x = 10;
}
```

## 集成

### VSCode 集成

```json
// .vscode/settings.json
{
    "[cnlang]": {
        "editor.defaultFormatter": "cnlang.cnfmt",
        "editor.formatOnSave": true
    }
}
```

### Git 钩子

```bash
#!/bin/bash
# .git/hooks/pre-commit

for file in $(git diff --cached --name-only --diff-filter=ACM | grep '\.cn$'); do
    cnfmt --check "$file" || exit 1
done
```

### CI/CD 集成

```yaml
# .github/workflows/format.yml
name: Format Check
on: [push, pull_request]
jobs:
  format:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Install cnfmt
        run: cmake --build build --target cnfmt
      - name: Check format
        run: cnfmt --check src/**/*.cn
```
