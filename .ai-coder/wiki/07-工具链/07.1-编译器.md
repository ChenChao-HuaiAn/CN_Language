# 编译器 (cnc)

## 概述

`cnc` 是 CN_Language 的主编译器，负责将 `.cn` 源代码编译为可执行文件或目标文件。

## 文件位置

[`src/cli/cnc/main.c`](../../src/cli/cnc/main.c:1)

## 使用方法

### 基本用法

```bash
# 编译为可执行文件
cnc 输入.cn -o 输出

# 编译并指定优化级别
cnc 输入.cn -O2 -o 输出

# 生成目标文件
cnc 输入.cn -c -o 输出.o

# 预处理后停止
cnc 输入.cn -E

# 编译为 C 代码
cnc 输入.cn --emit-c -o 输出.c
```

### 命令行选项

| 选项 | 描述 |
|------|------|
| `-o <文件>` | 指定输出文件 |
| `-c` | 只编译不链接 |
| `-E` | 只预处理 |
| `-S` | 只编译到汇编 |
| `-O0`, `-O1`, `-O2`, `-O3` | 优化级别 |
| `-g` | 生成调试信息 |
| `-I <目录>` | 添加头文件搜索路径 |
| `-L <目录>` | 添加库搜索路径 |
| `-l <库>` | 链接库 |
| `-D <宏>` | 定义宏 |
| `--emit-c` | 生成 C 代码 |
| `--freestanding` | Freestanding 模式 |
| `--target <三元组>` | 指定目标平台 |
| `-h`, `--help` | 显示帮助 |
| `-v`, `--version` | 显示版本 |

## 编译流程

```
源代码 → [预处理] → [词法分析] → [语法分析] → [语义分析] 
   → [IR 生成] → [优化] → [代码生成] → [汇编] → [链接] → 可执行文件
```

### 1. 预处理

```c
// 处理宏、条件编译、文件包含
CnPreprocessor pp;
cn_preprocessor_init(&pp, source, filename);
cn_preprocessor_run(&pp);
```

### 2. 词法分析

```c
// 将源代码转换为标记流
CnLexer lexer;
cn_frontend_lexer_init(&lexer, source, length, filename);
```

### 3. 语法分析

```c
// 构建抽象语法树
CnParser *parser = cn_frontend_parser_new(&lexer);
CnAstProgram *program;
cn_frontend_parse_program(parser, &program);
```

### 4. 语义分析

```c
// 名称解析和类型检查
CnSemScope *global_scope = cn_sem_scope_new(CN_SEM_SCOPE_GLOBAL, NULL);
cn_sem_resolve_names(global_scope, program, diagnostics);
cn_sem_check_types(global_scope, program, diagnostics);
```

### 5. IR 生成

```c
// 生成中间表示
CnIrModule *module = cn_ir_module_new();
CnIrGen gen;
cn_ir_gen_init(&gen, module);
cn_ir_gen_program(&gen, program);
```

### 6. 优化

```c
// 运行优化 passes
cn_ir_run_default_passes(module);
```

### 7. 代码生成

```c
// 生成 C 代码
CnCodeGen *codegen = cn_cgen_new();
cn_cgen_generate(codegen, module);
cn_cgen_emit(codegen, output_file);
```

## 输出格式

### 可执行文件

```bash
cnc 程序.cn -o 程序
./程序
```

### 目标文件

```bash
cnc 程序.cn -c -o 程序.o
gcc 程序.o -o 程序
```

### C 代码

```bash
cnc 程序.cn --emit-c -o 程序.c
gcc 程序.c -o 程序
```

## 优化级别

### -O0（无优化）

```bash
cnc 程序.cn -O0 -o 程序
```

- 保留所有调试信息
- 编译速度最快
- 运行速度最慢

### -O1（基本优化）

```bash
cnc 程序.cn -O1 -o 程序
```

- 常量折叠
- 死代码消除
- 基本块重排

### -O2（标准优化）

```bash
cnc 程序.cn -O2 -o 程序
```

- 内联小函数
- 循环优化
- 指令调度

### -O3（激进优化）

```bash
cnc 程序.cn -O3 -o 程序
```

- 激进内联
- 向量化
- 预取优化

## 错误处理

### 编译错误

```
错误：在文件 test.cn 第 10 行第 5 列
  未定义的标识符 '未定义变量'
```

### 类型错误

```
错误：在文件 test.cn 第 15 行第 10 列
  类型不兼容：无法将 '字符串' 赋值给 '整数'
```

### 链接错误

```
错误：未定义的符号 '外部函数'
  引用自：test.o
```

## 使用示例

### Hello World

```cn
// hello.cn
函数 主程序 () {
    打印 ("你好，世界！\n");
    返回 0;
}
```

```bash
cnc hello.cn -o hello
./hello
```

### 多文件编译

```cn
// 数学.cn
模块 数学;

导出 函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}
```

```cn
// 主程序.cn
模块 主程序;

从 数学 导入 加法;

函数 主程序 () {
    整数 结果 = 加法 (5, 3);
    打印 ("结果：%d\n", 结果);
    返回 0;
}
```

```bash
cnc 数学.cn 主程序.cn -o 程序
./程序
```

### Freestanding 编译

```cn
#定义 自由站立

模块 内核;

函数 内核入口 () {
    // 内核代码
}
```

```bash
cnc 内核.cn --freestanding -o 内核.bin
```
