# 并发支持

## 概述

CN_Language 运行时提供并发编程支持，包括互斥锁、读写锁、自旋锁和条件变量。

## 文件位置

[`src/runtime/sync/sync.c`](../../src/runtime/sync/sync.c:1)

## 互斥锁 (Mutex)

### 概述

互斥锁是最基本的同步原语，用于保护共享资源免受并发访问。

### API

```c
// 创建互斥锁
CnMutex* cn_mutex_create(void);

// 销毁互斥锁
void cn_mutex_destroy(CnMutex *mutex);

// 加锁
void cn_mutex_lock(CnMutex *mutex);

// 尝试加锁（不阻塞）
int cn_mutex_trylock(CnMutex *mutex);

// 解锁
void cn_mutex_unlock(CnMutex *mutex);
```

### 使用示例

```cn
// 创建互斥锁
互斥锁* 锁 = 同步。创建互斥锁 ();

整数 共享计数 = 0;

函数 线程安全增加 () {
    同步。互斥锁加锁 (锁);
    共享计数 = 共享计数 + 1;
    同步。互斥锁解锁 (锁);
}

// 使用完毕后销毁
同步。销毁互斥锁 (锁);
```

### RAII 风格

```cn
函数 线程安全操作 () {
    互斥锁 锁 = 同步。创建互斥锁 ();
    
    // 使用 defer 自动释放
    延迟 { 同步。销毁互斥锁 (锁); };
    
    同步。互斥锁加锁 (锁);
    延迟 { 同步。互斥锁解锁 (锁); };
    
    // 临界区代码
    执行线程安全操作 ();
}
```

## 读写锁 (RWLock)

### 概述

读写锁允许多个读者同时访问，但写者独占访问。适合读多写少的场景。

### API

```c
// 创建读写锁
CnRWLock* cn_rwlock_create(void);

// 销毁读写锁
void cn_rwlock_destroy(CnRWLock *lock);

// 读锁
void cn_rwlock_rdlock(CnRWLock *lock);

// 写锁
void cn_rwlock_wrlock(CnRWLock *lock);

// 解锁
void cn_rwlock_unlock(CnRWLock *lock);
```

### 使用示例

```cn
// 创建读写锁
读写锁* 锁 = 同步。创建读写锁 ();

字符串 共享数据;

函数 读取数据 () {
    同步。读写锁读加锁 (锁);
    // 可以多个读者同时读取
    打印 ("%s\n", 共享数据);
    同步。读写锁解锁 (锁);
}

函数 写入数据 (字符串 新数据) {
    同步。读写锁写加锁 (锁);
    // 写者独占
    共享数据 = 新数据;
    同步。读写锁解锁 (锁);
}
```

## 自旋锁 (Spinlock)

### 概述

自旋锁在等待时不阻塞，而是循环检查锁状态。适合短临界区。

### API

```c
// 创建自旋锁
CnSpinlock* cn_spinlock_create(void);

// 销毁自旋锁
void cn_spinlock_destroy(CnSpinlock *lock);

// 加锁
void cn_spinlock_lock(CnSpinlock *lock);

// 解锁
void cn_spinlock_unlock(CnSpinlock *lock);
```

### 使用示例

```cn
// 创建自旋锁
自旋锁* 锁 = 同步。创建自旋锁 ();

整数 计数器 = 0;

函数 原子增加 () {
    同步。自旋锁加锁 (锁);
    计数器 = 计数器 + 1;
    同步。自旋锁解锁 (锁);
}
```

### 性能对比

| 锁类型 | 等待方式 | 适用场景 |
|--------|----------|----------|
| 互斥锁 | 阻塞 | 长临界区 |
| 自旋锁 | 循环等待 | 短临界区 |
| 读写锁 | 混合 | 读多写少 |

## 条件变量 (Condition Variable)

### 概述

条件变量用于线程间通信，允许线程等待某个条件成立。

### API

```c
// 创建条件变量
CnCondvar* cn_condvar_create(void);

// 销毁条件变量
void cn_condvar_destroy(CnCondvar *cond);

// 等待
void cn_condvar_wait(CnCondvar *cond, CnMutex *mutex);

// 限时等待
int cn_condvar_wait_timeout(CnCondvar *cond, CnMutex *mutex, int timeout_ms);

// 唤醒一个等待线程
void cn_condvar_signal(CnCondvar *cond);

// 唤醒所有等待线程
void cn_condvar_broadcast(CnCondvar *cond);
```

### 使用示例

```cn
互斥锁* 锁 = 同步。创建互斥锁 ();
条件变量* 条件 = 同步。创建条件变量 ();

布尔 数据就绪 = 假;
整数 共享数据;

函数 生产者 () {
    同步。互斥锁加锁 (锁);
    
    // 生产数据
    共享数据 = 计算数据 ();
    数据就绪 = 真;
    
    // 唤醒消费者
    同步。条件变量通知 (条件);
    
    同步。互斥锁解锁 (锁);
}

函数 消费者 () {
    同步。互斥锁加锁 (锁);
    
    // 等待数据
    当 (!数据就绪) {
        同步。条件变量等待 (条件，锁);
    }
    
    // 消费数据
    使用数据 (共享数据);
    数据就绪 = 假;
    
    同步。互斥锁解锁 (锁);
}
```

### 生产者 - 消费者模式

```cn
#定义 缓冲区大小 10

整数 缓冲区 [缓冲区大小];
整数 头 = 0;
整数 尾 = 0;
整数 计数 = 0;

互斥锁* 锁 = 同步。创建互斥锁 ();
条件变量* 非空 = 同步。创建条件变量 ();
条件变量* 非满 = 同步。创建条件变量 ();

函数 生产 (整数 数据) {
    同步。互斥锁加锁 (锁);
    
    // 等待缓冲区非满
    当 (计数 == 缓冲区大小) {
        同步。条件变量等待 (非满，锁);
    }
    
    // 放入数据
    缓冲区 [尾] = 数据;
    尾 = (尾 + 1) % 缓冲区大小;
    计数 = 计数 + 1;
    
    // 通知消费者
    同步。条件变量通知 (非空);
    
    同步。互斥锁解锁 (锁);
}

函数 消费 () {
    同步。互斥锁加锁 (锁);
    
    // 等待缓冲区非空
    当 (计数 == 0) {
        同步。条件变量等待 (非空，锁);
    }
    
    // 取出数据
    整数 数据 = 缓冲区 [头];
    头 = (头 + 1) % 缓冲区大小;
    计数 = 计数 - 1;
    
    // 通知生产者
    同步。条件变量通知 (非满);
    
    同步。互斥锁解锁 (锁);
    
    返回 数据;
}
```

## 原子操作

### 概述

原子操作是不可中断的操作，用于实现无锁数据结构。

### API

```c
// 原子加载
整数 原子。加载 (整数* ptr);

// 原子存储
void 原子。存储 (整数* ptr, 整数 value);

// 原子交换
整数 原子。交换 (整数* ptr, 整数 value);

// 原子比较交换
整数 原子。比较交换 (整数* ptr, 整数 expected, 整数 desired);

// 原子增加
整数 原子。增加 (整数* ptr);

// 原子减少
整数 原子。减少 (整数* ptr);

// 原子加法
整数 原子。加 (整数* ptr, 整数 value);

// 原子减法
整数 原子。减 (整数* ptr, 整数 value);
```

### 使用示例

```cn
整数 原子计数器;

函数 线程安全增加 () {
    原子。增加 (&原子计数器);
}

函数 获取计数 () {
    返回 原子。加载 (&原子计数器);
}
```

### 无锁栈示例

```cn
结构体 节点 {
    整数 数据;
    节点* 下一个;
}

结构体 无锁栈 {
    节点* 头;
}

函数 无锁栈推入 (无锁栈* 栈，节点* 新节点) {
    循环 {
        节点* 旧头 = 原子。加载 (&栈->头);
        新节点->下一个 = 旧头;
        
        如果 (原子。比较交换 (&栈->头，旧头，新节点) == 旧头) {
            返回;  // 成功
        }
        // 失败，重试
    }
}

函数 无锁栈弹出 (无锁栈* 栈) {
    循环 {
        节点* 旧头 = 原子。加载 (&栈->头);
        
        如果 (旧头 == 空) {
            返回 空;  // 栈空
        }
        
        节点* 新头 = 旧头->下一个;
        
        如果 (原子。比较交换 (&栈->头，旧头，新头) == 旧头) {
            返回 旧头;  // 成功
        }
        // 失败，重试
    }
}
```

## 屏障 (Barrier)

### 概述

屏障用于同步多个线程，所有线程必须到达屏障后才能继续。

### API

```c
// 创建屏障
CnBarrier* cn_barrier_create(int count);

// 销毁屏障
void cn_barrier_destroy(CnBarrier *barrier);

// 等待屏障
void cn_barrier_wait(CnBarrier *barrier);
```

### 使用示例

```cn
#定义 线程数 4

屏障* 屏障 = 同步。创建屏障 (线程数);

函数 工作线程 () {
    // 阶段 1
    执行阶段 1();
    
    // 等待所有线程完成阶段 1
    同步。屏障等待 (屏障);
    
    // 阶段 2
    执行阶段 2();
    
    // 等待所有线程完成阶段 2
    同步。屏障等待 (屏障);
}
```

## 信号量 (Semaphore)

### 概述

信号量用于控制同时访问资源的线程数量。

### API

```c
// 创建信号量
CnSemaphore* cn_semaphore_create(int initial_value);

// 销毁信号量
void cn_semaphore_destroy(CnSemaphore *sem);

// P 操作（等待）
void cn_semaphore_wait(CnSemaphore *sem);

// V 操作（信号）
void cn_semaphore_post(CnSemaphore *sem);
```

### 使用示例

```cn
// 限制同时访问数据库连接的线程数
信号量* 数据库信号量 = 同步。创建信号量 (5);  // 最多 5 个连接

函数 访问数据库 () {
    同步。信号量等待 (数据库信号量);  // 获取许可
    
    // 访问数据库...
    
    同步。信号量发布 (数据库信号量);  // 释放许可
}
```
