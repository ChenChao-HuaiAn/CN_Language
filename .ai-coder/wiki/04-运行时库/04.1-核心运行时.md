# 核心运行时

## 概述

核心运行时提供 CN_Language 程序运行所需的基础功能，包括初始化、退出处理和基础打印函数。

## 文件位置

[`src/runtime/core/runtime.c`](../../src/runtime/core/runtime.c:1)

## 运行时状态

```c
typedef struct CnRuntimeState {
    int exit_code;                 // 退出码
    size_t total_allocations;      // 总分配次数
    size_t total_freed;            // 总释放次数
    // ... 其他状态
} CnRuntimeState;

static CnRuntimeState g_rt_state;  // 全局运行时状态
```

## 初始化和退出

### 运行时初始化

```c
void cn_rt_init(void);
```

**功能：**
1. 初始化全局运行时状态
2. 根据编译模式初始化内存子系统
3. 注册 I/O 回调（如果适用）

**示例：**
```c
// 在 main 函数开始时自动调用
函数 主程序 () {
    // 运行时已初始化
    打印 ("你好");
}
```

### 运行时退出

```c
void cn_rt_exit(void);
```

**功能：**
1. 执行清理操作
2. 输出内存统计信息（如果启用）
3. 调用注册的退出处理程序

## Freestanding 支持

### 内置函数

在 freestanding 模式下，提供以下内置函数：

```c
// 内存操作
void* cn_rt_freestanding_memcpy(void *dest, const void *src, size_t n);
void* cn_rt_freestanding_memset(void *s, int c, size_t n);

// 字符串操作
size_t cn_rt_freestanding_strlen(const char *s);
char* cn_rt_freestanding_strcpy(char *dest, const char *src);
```

### 内核 I/O 回调

```c
// 注册内核 I/O 回调
void cn_rt_kernel_io_puts(const char* str);
void cn_rt_kernel_io_print_int(long long val);
void cn_rt_kernel_io_putchar(char c);
```

**示例（内核开发）：**
```cn
模块 内核;

函数 内核入口 () {
    // 注册 I/O 回调
    运行时。注册 Io 回调 (内核_输出，内核_输出整数，内核_输出字符);
    
    打印 ("内核启动...\n");
}

函数 内核_输出 (字符串 文本) {
    // 直接写入 VGA 缓冲区或串口
    硬件。写端口 (0x3F8, 文本);
}
```

## 内存分配

### 基础分配

```c
void* cn_rt_malloc(size_t size);
void* cn_rt_calloc(size_t count, size_t size);
void* cn_rt_realloc(void* ptr, size_t new_size);
void cn_rt_free(void* ptr);
```

**示例：**
```cn
// 分配内存
整数* 数据 = 内存。分配 (sizeof(整数) * 100);

// 清零分配
整数* 清零数据 = 内存。分配清零 (100, sizeof(整数));

// 重新分配
数据 = 内存。重新分配 (数据，sizeof(整数) * 200);

// 释放
内存。释放 (数据);
```

### 自定义分配器

```c
typedef struct CnRtMemoryConfig {
    void* (*alloc_func)(size_t size, void* user_data);
    void  (*free_func)(void* ptr, void* user_data);
    void* (*realloc_func)(void* ptr, size_t size, void* user_data);
    void* user_data;
} CnRtMemoryConfig;

void cn_rt_memory_init(CnRtMemoryConfig* config);
```

**示例：**
```c
// 自定义 Arena 分配器
void* arena_alloc(size_t size, void* user_data) {
    CnArena *arena = (CnArena*)user_data;
    return cn_arena_alloc(arena, size);
}

void arena_free(void* ptr, void* user_data) {
    // Arena 分配器通常不单独释放
}

// 初始化
CnRtMemoryConfig config = {
    .alloc_func = arena_alloc,
    .free_func = arena_free,
    .user_data = my_arena
};
cn_rt_memory_init(&config);
```

## 内存统计

### 获取统计信息

```c
typedef struct CnRtMemoryStats {
    size_t total_allocated_bytes;  // 总分配字节数
    size_t current_allocated_bytes; // 当前分配字节数
    size_t allocation_count;        // 分配次数
    size_t free_count;              // 释放次数
} CnRtMemoryStats;

void cn_rt_memory_get_stats(CnRtMemoryStats* stats);
```

### 输出统计

```c
void cn_rt_memory_dump_stats(void);
```

**输出示例：**
```
--- CN Runtime Memory Stats ---
Total Allocated: 1048576 bytes
Allocations: 1024
Frees: 1000
-------------------------------
```
