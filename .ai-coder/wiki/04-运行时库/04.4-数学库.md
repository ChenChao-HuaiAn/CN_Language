# 数学库

## 概述

CN_Language 运行时提供基础数学函数，包括算术运算、三角函数、指数对数等。

## 文件位置

[`src/runtime/math/math.c`](../../src/runtime/math/math.c:1)

## 使用示例

### 计算三角形面积（海伦公式）

```cn
函数 三角形面积 (小数 a, 小数 b, 小数 c) {
    小数 s = (a + b + c) / 2;  // 半周长
    返回 数学。sqrt(s * (s - a) * (s - b) * (s - c));
}
```

### 计算两点距离

```cn
函数 距离 (小数 x1, 小数 y1, 小数 x2, 小数 y2) {
    小数 dx = x2 - x1;
    小数 dy = y2 - y1;
    返回 数学。sqrt(dx * dx + dy * dy);
}
```

### 角度转换

```cn
函数 弧度转角度 (小数 弧度) {
    返回 弧度 * 180 / 数学。PI;
}

函数 角度转弧度 (小数 角度) {
    返回 角度 * 数学。PI / 180;
}
```
