# CN_Language 核心特性

## 语言特性

### 基础类型

CN_Language 提供以下基础数据类型：

| 类型 | 描述 | 示例 |
|------|------|------|
| `整数` | 整型数值 | `整数 x = 10;` |
| `小数` | 浮点型数值 | `小数 y = 3.14;` |
| `布尔` | 布尔值（真/假） | `布尔 b = 真;` |
| `字符串` | 字符串类型 | `字符串 s = "你好";` |
| `字符` | 单个字符 | `字符 c = 'A';` |
| `空` | 空类型 | `空 结果;` |

### 复合类型

#### 数组

```cn
// 固定大小数组
整数 数组 [10] 数字;

// 初始化数组
整数 数组 [5] 分数 = {1, 2, 3, 4, 5};

// 多维数组
整数 数组 [3][3] 矩阵;
```

#### 结构体

```cn
结构体 学生 {
    字符串 姓名;
    整数 年龄;
    小数 成绩;
}

函数 使用学生 () {
    学生 小明 = 学生 {姓名："小明", 年龄：18, 成绩：95.5};
    打印 (小明。姓名);
}
```

#### 枚举

```cn
枚举 颜色 {
    红，
    绿，
    蓝
}

函数 使用枚举 () {
    颜色 c = 颜色。红;
    如果 (c == 颜色。红) {
        打印 ("红色");
    }
}
```

### 指针支持

```cn
// 普通指针
整数 x = 10;
整数* 指针 = &x;
打印 (*指针);  // 输出 10

// 函数指针
函数 类型 加法 (整数 a, 整数 b) {
    返回 a + b;
}

函数 类型 (*操作指针) (整数，整数) = 加法;
整数 结果 = 操作指针 (5, 3);  // 结果 = 8
```

## 控制流

### 条件语句

```cn
// 如果 - 否则
整数 分数 = 85;

如果 (分数 >= 90) {
    打印 ("优秀");
} 否则 如果 (分数 >= 60) {
    打印 ("及格");
} 否则 {
    打印 ("不及格");
}

// 选择语句
整数 选项 = 2;

选择 (选项) {
    情况 1:
        打印 ("选项一");
        中断;
    情况 2:
        打印 ("选项二");
        中断;
    默认:
        打印 ("未知选项");
}
```

### 循环语句

```cn
// 当循环
整数 i = 0;
当 (i < 10) {
    打印 (i);
    i = i + 1;
}

// 循环语句 (for)
循环 (整数 j = 0; j < 10; j = j + 1) {
    打印 (j);
}

// 中断和继续
循环 (整数 k = 0; k < 100; k = k + 1) {
    如果 (k == 50) {
        中断;  // 跳出循环
    }
    如果 (k % 2 == 0) {
        继续;  // 跳过本次迭代
    }
    打印 (k);
}
```

## 函数系统

### 函数定义

```cn
// 无参数无返回值
函数 打招呼 () {
    打印 ("你好！");
}

// 带参数和返回值
函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

// 多参数
函数 计算 (整数 x, 整数 y, 整数 z) {
    返回 x + y + z;
}
```

### 函数参数

```cn
// 值传递
函数 值传递 (整数 x) {
    x = x + 1;  // 不影响原值
}

// 引用传递（指针）
函数 引用传递 (整数* x) {
    *x = *x + 1;  // 影响原值
}

// 数组参数
函数 处理数组 (整数 数组 [] 数据，整数 大小) {
    循环 (整数 i = 0; i < 大小; i = i + 1) {
        打印 (数据 [i]);
    }
}
```

### 递归函数

```cn
函数 阶乘 (整数 n) {
    如果 (n <= 1) {
        返回 1;
    }
    返回 n * 阶乘 (n - 1);
}

函数 斐波那契 (整数 n) {
    如果 (n <= 1) {
        返回 n;
    }
    返回 斐波那契 (n - 1) + 斐波那契 (n - 2);
}
```

## 模块系统

### 模块声明

```cn
// 模块文件：数学模块.cn
模块 数学模块;

函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

函数 减法 (整数 a, 整数 b) {
    返回 a - b;
}

// 导出特定符号
导出 函数 乘法 (整数 a, 整数 b) {
    返回 a * b;
}
```

### 模块导入

```cn
// 导入整个模块
导入 数学模块;

整数 结果 = 数学模块。加法 (5, 3);

// 选择性导入
从 数学模块 导入 加法，减法;

整数 和 = 加法 (10, 20);

// 别名导入
从 数学模块 导入 乘法 作为 乘;

整数 积 = 乘 (5, 6);
```

### 可见性控制

```cn
// 默认私有（模块内可见）
函数 内部辅助 () {
    // 只能在模块内访问
}

// 显式导出（对外公开）
导出 函数 公开接口 () {
    // 可以被其他模块访问
}

// 导出类型
导出 结构体 公开结构 {
    整数 字段;
}
```

// 安全的内存复制

内存。设置 (缓冲区，值，大小);

// 边界检查
如果 (索引 < 数组大小) {

## 系统编程支持

### Freestanding 模式

```cn
// 无标准库依赖
#定义 自由站立

模块 内核;

函数 内核入口 () {
    // 直接使用硬件
    硬件。写端口 (0x3F8, 'A');
}
```

### 中断处理

```cn
// 注册中断处理程序
运行时。注册中断处理程序 (中断号，处理函数);

函数 定时器处理 () {
    // 定时器中断处理
    硬件。清除中断 (中断号);
}
```

### 内联汇编

```cn
// 内联汇编（通过 C 后端）
#包含 <内联汇编>

函数 底层操作 () {
    // 通过 C 代码嵌入汇编
    asm("mov eax, 1");
    asm("int 0x80");
}
```
