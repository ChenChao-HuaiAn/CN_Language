# 前端 API

## 概述

前端 API 提供词法分析、语法分析、AST 操作和语义分析功能。

## 词法分析

### CnLexer 结构

```c
typedef struct CnLexer {
    const char *source;      // 源代码
    size_t length;           // 长度
    size_t offset;           // 当前位置
    size_t line;             // 当前行
    size_t column;           // 当前列
    const char *filename;    // 文件名
    CnDiagnostics *diagnostics;  // 诊断
} CnLexer;
```

### 函数

```c
// 初始化词法分析器
void cn_frontend_lexer_init(CnLexer *lexer, 
                            const char *source, 
                            size_t length, 
                            const char *filename);

// 获取下一个标记
int cn_frontend_lexer_next_token(CnLexer *lexer, CnToken *token);

// 获取当前标记
CnToken cn_frontend_lexer_current_token(CnLexer *lexer);

// 跳过空白
void cn_frontend_lexer_skip_whitespace(CnLexer *lexer);
```

### 使用示例

```c
#include "cnlang/frontend/lexer.h"

CnLexer lexer;
cn_frontend_lexer_init(&lexer, source, strlen(source), "test.cn");

CnToken token;
while (cn_frontend_lexer_next_token(&lexer, &token)) {
    printf("Token: %d at line %zu\n", token.kind, token.line);
}
```

## 语法分析

### CnParser 结构

```c
typedef struct CnParser CnParser;
```

### 函数

```c
// 创建语法分析器
CnParser *cn_frontend_parser_new(CnLexer *lexer);

// 销毁语法分析器
void cn_frontend_parser_free(CnParser *parser);

// 设置诊断
void cn_frontend_parser_set_diagnostics(CnParser *parser, 
                                         CnDiagnostics *diagnostics);

// 解析程序
bool cn_frontend_parse_program(CnParser *parser, CnAstProgram **out_program);
```

### 使用示例

```c
#include "cnlang/frontend/parser.h"

CnParser *parser = cn_frontend_parser_new(&lexer);
CnAstProgram *program;

if (cn_frontend_parse_program(parser, &program)) {
    // 解析成功
} else {
    // 解析失败
}

cn_frontend_parser_free(parser);
```

### 表达式类型

```c
typedef enum {
    CN_AST_EXPR_BINARY,           // 二元运算
    CN_AST_EXPR_LOGICAL,          // 逻辑运算
    CN_AST_EXPR_UNARY,            // 一元运算
    CN_AST_EXPR_ASSIGN,           // 赋值
    CN_AST_EXPR_CALL,             // 函数调用
    CN_AST_EXPR_IDENTIFIER,       // 标识符
    CN_AST_EXPR_INTEGER_LITERAL,  // 整数
    CN_AST_EXPR_FLOAT_LITERAL,    // 浮点数
    CN_AST_EXPR_STRING_LITERAL,   // 字符串
    CN_AST_EXPR_BOOL_LITERAL,     // 布尔
    CN_AST_EXPR_ARRAY_LITERAL,    // 数组
    CN_AST_EXPR_INDEX,            // 索引
    CN_AST_EXPR_MEMBER_ACCESS,    // 成员访问
    CN_AST_EXPR_STRUCT_LITERAL,   // 结构体字面量
} CnAstExprKind;
```

### 语句类型

```c
typedef enum {
    CN_AST_STMT_BLOCK,            // 代码块
    CN_AST_STMT_VAR_DECL,         // 变量声明
    CN_AST_STMT_EXPR,             // 表达式语句
    CN_AST_STMT_RETURN,           // 返回
    CN_AST_STMT_IF,               // 条件
    CN_AST_STMT_WHILE,            // while 循环
    CN_AST_STMT_FOR,              // for 循环
    CN_AST_STMT_SWITCH,           // switch
    CN_AST_STMT_BREAK,            // break
    CN_AST_STMT_CONTINUE,         // continue
    CN_AST_STMT_STRUCT_DECL,      // 结构体声明
    CN_AST_STMT_ENUM_DECL,        // 枚举声明
    CN_AST_STMT_MODULE_DECL,      // 模块声明
    CN_AST_STMT_IMPORT,           // 导入
} CnAstStmtKind;
```
