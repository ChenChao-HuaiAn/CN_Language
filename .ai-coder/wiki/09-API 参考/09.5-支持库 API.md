# 支持库 API

## 概述

支持库 API 提供通用工具函数，包括字符串处理、容器、内存工具等。

## 字符串工具

### 字符串构建器

```c
// 创建
CnStringBuilder* cn_string_builder_new(void);
CnStringBuilder* cn_string_builder_new_with_capacity(size_t capacity);

// 销毁
void cn_string_builder_free(CnStringBuilder *sb);

// 追加
void cn_string_builder_append(CnStringBuilder *sb, const char *str);
void cn_string_builder_append_char(CnStringBuilder *sb, char c);
void cn_string_builder_append_format(CnStringBuilder *sb, 
                                      const char *format, ...);

// 获取字符串
const char* cn_string_builder_cstr(CnStringBuilder *sb);
char* cn_string_builder_to_string(CnStringBuilder *sb);

// 清空
void cn_string_builder_clear(CnStringBuilder *sb);

// 长度
size_t cn_string_builder_length(CnStringBuilder *sb);
```

### 字符串工具函数

```c
// 分割
char** cn_string_split(const char *str, const char *delimiter, int *count);

// 连接
char* cn_string_join(const char **parts, int count, const char *delimiter);

// 修剪
char* cn_string_trim(const char *str);
char* cn_string_trim_left(const char *str);
char* cn_string_trim_right(const char *str);

// 查找
int cn_string_contains(const char *haystack, const char *needle);
int cn_string_starts_with(const char *str, const char *prefix);
int cn_string_ends_with(const char *str, const char *suffix);

// 替换
char* cn_string_replace(const char *str, 
                        const char *old, 
                        const char *new);

// 转义
char* cn_string_escape(const char *str);
char* cn_string_unescape(const char *str);
```
