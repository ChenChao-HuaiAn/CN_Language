# 模块语义

## 概述

模块语义处理 CN_Language 的模块系统，包括模块声明、导入导出、可见性控制和依赖关系。

## 文件位置

[`src/semantics/resolution/module_semantics.c`](../../src/semantics/resolution/module_semantics.c:1)

## 模块声明

### 语法

```cn
模块 模块名;

// 子模块
模块 父模块。子模块;
```

### 语义处理

```c
void process_module_decl(CnSemScope *global_scope, CnAstModuleDecl *decl) {
    // 创建模块作用域
    CnSemScope *module_scope = cn_sem_scope_new(CN_SEM_SCOPE_FILE_MODULE, global_scope);
    cn_sem_scope_set_name(module_scope, decl->name, decl->name_length);
    
    // 插入模块符号到全局作用域
    CnSemSymbol *module_sym = cn_sem_scope_insert_symbol(global_scope,
                                                          decl->name,
                                                          decl->name_length,
                                                          CN_SEM_SYMBOL_MODULE);
    module_sym->as.module.module_scope = module_scope;
    
    // 处理模块内声明
    for (size_t i = 0; i < decl->decl_count; i++) {
        process_module_member(module_scope, decl->decls[i]);
    }
}
```

### 语义处理

```c
void process_import_stmt(CnSemScope *current_scope, 
                         CnAstImportStmt *import,
                         CnDiagnostics *diagnostics) {
    // 解析模块路径
    CnModule *module = resolve_module_path(import->module_path);
    
    if (!module) {
        report_error(diagnostics, "模块未找到：%.*s",
                     (int)import->module_path_length,
                     import->module_path);
        return;
    }
    
    // 处理导入成员
    if (import->is_selective) {
        // 选择性导入
        for (size_t i = 0; i < import->member_count; i++) {
            CnImportMember *member = &import->members[i];
            
            // 在模块作用域中查找符号
            CnSemSymbol *sym = cn_sem_scope_lookup(module->scope,
                                                    member->name,
                                                    member->name_length);
            
            if (!sym) {
                report_error(diagnostics, 
                             "模块 '%.*s' 中没有成员 '%.*s'",
                             (int)import->module_path_length,
                             import->module_path,
                             (int)member->name_length,
                             member->name);
                continue;
            }
            
            // 检查可见性
            if (!sym->is_public) {
                report_error(diagnostics,
                             "符号 '%.*s' 是私有的，不能导入",
                             (int)member->name_length,
                             member->name);
                continue;
            }
            
            // 插入到当前作用域
            const char *local_name = member->alias ? member->alias : member->name;
            cn_sem_scope_insert_symbol(current_scope,
                                        local_name,
                                        strlen(local_name),
                                        sym->kind);
        }
    } else {
        // 完整导入：将模块符号插入当前作用域
        cn_sem_scope_insert_symbol(current_scope,
                                    import->module_path,
                                    import->module_path_length,
                                    CN_SEM_SYMBOL_MODULE);
    }
}
```

## 导出声明

### 显式导出

```cn
模块 我的模块;

// 显式导出函数
导出 函数 公开函数 () {
    // ...
}

// 私有函数（默认）
函数 私有函数 () {
    // 只能在模块内访问
}
```

### 批量导出

```cn
#公开  // 设置默认可见性为公开

函数 函数 1() { }  // 自动导出
函数 函数 2() { }  // 自动导出

#私有  // 恢复默认为私有

函数 内部函数 () { }  // 不导出
```

### 导出结构体和枚举

```cn
// 导出结构体
导出 结构体 公开结构 {
    整数 公开字段;
    整数 私有字段;  // 字段也可以有独立可见性
}

// 导出枚举
导出 枚举 公开枚举 {
    成员 1,
    成员 2
}
```

### 语义处理

```c
void process_export_decl(CnSemScope *module_scope, CnAstDecl *decl) {
    switch (decl->kind) {
        case CN_AST_DECL_FUNCTION: {
            CnAstFunctionDecl *func = &decl->as.function;
            
            // 在模块作用域中查找函数符号
            CnSemSymbol *sym = cn_sem_scope_lookup_shallow(module_scope,
                                                            func->name,
                                                            func->name_length);
            if (sym) {
                sym->is_public = 1;  // 标记为公开
            }
            break;
        }
        case CN_AST_DECL_STRUCT: {
            CnAstStructDecl *struct_decl = &decl->as.struct_decl;
            
            CnSemSymbol *sym = cn_sem_scope_lookup_shallow(module_scope,
                                                            struct_decl->name,
                                                            struct_decl->name_length);
            if (sym) {
                sym->is_public = 1;
                sym->type->as.struct_type.is_public = 1;
            }
            break;
        }
        // 处理其他类型...
    }
}
```

## 可见性控制

### 可见性级别

| 级别 | 关键字 | 描述 |
|------|--------|------|
| 私有 | `#私有` / 默认 | 仅模块内可见 |
| 公开 | `#公开` | 对所有模块可见 |
| 导出 | `导出` | 显式导出单个符号 |

### 可见性检查

```c
int check_symbol_visibility(CnSemSymbol *sym, CnSemScope *access_scope) {
    if (!sym) return 0;
    
    // 公开符号总是可访问
    if (sym->is_public) return 1;
    
    // 检查是否在同一模块
    CnSemScope *decl_scope = sym->decl_scope;
    while (decl_scope) {
        if (decl_scope->kind == CN_SEM_SCOPE_FILE_MODULE) {
            CnSemScope *current_module = get_current_module_scope(access_scope);
            if (decl_scope == current_module) {
                return 1;  // 同一模块内可访问
            }
            break;
        }
        decl_scope = decl_scope->parent;
    }
    
    return 0;  // 不可访问
}
```

### 可见性错误

```c
void check_member_access(CnSemSymbol *member_sym, 
                         const char *accessor_name,
                         CnDiagnostics *diagnostics) {
    if (!member_sym->is_public) {
        report_error(diagnostics,
                     "符号 '%.*s' 是私有的，不能从 '%s' 访问",
                     (int)member_sym->name_length,
                     member_sym->name,
                     accessor_name);
    }
}
```

## 模块依赖

### 依赖分析

```c
typedef struct CnModuleDependency {
    CnModule *from_module;
    CnModule *to_module;
    CnAstImportStmt *import_stmt;
} CnModuleDependency;

// 构建依赖图
void build_dependency_graph(CnModule **modules, size_t count) {
    for (size_t i = 0; i < count; i++) {
        CnModule *module = modules[i];
        
        for (size_t j = 0; j < module->import_count; j++) {
            CnAstImportStmt *import = module->imports[j];
            CnModule *imported = resolve_module_path(import->module_path);
            
            if (imported) {
                add_dependency(module, imported);
            }
        }
    }
}
```

### 循环依赖检测

```c
int detect_circular_dependency(CnModule *start, CnModule *current, 
                               CnHashSet *visited) {
    if (current == start) {
        return 1;  // 发现循环
    }
    
    if (hash_set_contains(visited, current)) {
        return 0;  // 已访问，但不是循环
    }
    
    hash_set_add(visited, current);
    
    for (size_t i = 0; i < current->dependency_count; i++) {
        if (detect_circular_dependency(start, 
                                        current->dependencies[i], 
                                        visited)) {
            return 1;
        }
    }
    
    hash_set_remove(visited, current);
    return 0;
}
```

### 依赖错误

```c
void report_circular_dependency(CnDiagnostics *diagnostics,
                                CnModule **cycle,
                                size_t cycle_length) {
    char *message = build_cycle_message(cycle, cycle_length);
    
    cn_support_diagnostics_report(diagnostics,
                                  CN_DIAG_SEVERITY_ERROR,
                                  CN_DIAG_CODE_SEM_CIRCULAR_DEPENDENCY,
                                  "模块依赖",
                                  0, 0,
                                  "检测到循环依赖：%s",
                                  message);
    
    free(message);
}
```

## 使用示例

### 模块定义和使用

```cn
// 文件：数学/基础.cn
模块 数学。基础;

// 私有辅助函数
函数 验证参数 (整数 a, 整数 b) {
    返回 a >= 0 && b >= 0;
}

// 公开函数
导出 函数 加法 (整数 a, 整数 b) {
    如果 (!验证参数 (a, b)) {
        返回 -1;
    }
    返回 a + b;
}

导出 函数 减法 (整数 a, 整数 b) {
    返回 a - b;
}
```

```cn
// 文件：主程序.cn
模块 主程序;

// 导入方式 1：完整导入
导入 数学。基础;

函数 使用数学 1() {
    整数 结果 = 数学。基础。加法 (5, 3);
}

// 导入方式 2：选择性导入
从 数学。基础 导入 加法，减法;

函数 使用数学 2() {
    整数 和 = 加法 (10, 20);
    整数 差 = 减法 (50, 30);
}

// 导入方式 3：别名导入
从 数学。基础 导入 加法 作为 加;

函数 使用数学 3() {
    整数 结果 = 加 (100, 200);
}
```
