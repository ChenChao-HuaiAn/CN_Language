# 类型检查

## 概述

类型检查验证程序中的类型使用是否正确，包括赋值兼容性、运算合法性和函数调用正确性。

## 文件位置

[`src/semantics/checker/semantic_passes.c`](../../src/semantics/checker/semantic_passes.c:1)

## 类型检查 Pass

### 入口函数

```c
bool cn_sem_check_types(CnSemScope *global_scope,
                        CnAstProgram *program,
                        CnDiagnostics *diagnostics);
```

**功能：**
1. 检查所有函数的类型
2. 检查所有语句的类型
3. 检查所有表达式的类型
4. 报告类型错误

## 表达式类型检查

### 字面量

```c
CnType *check_literal_type(CnAstExpr *expr) {
    switch (expr->kind) {
        case CN_AST_EXPR_INTEGER_LITERAL:
            return cn_type_new_primitive(CN_TYPE_INT);
        case CN_AST_EXPR_FLOAT_LITERAL:
            return cn_type_new_primitive(CN_TYPE_FLOAT);
        case CN_AST_EXPR_STRING_LITERAL:
            return cn_type_new_primitive(CN_TYPE_STRING);
        case CN_AST_EXPR_BOOL_LITERAL:
            return cn_type_new_primitive(CN_TYPE_BOOL);
        default:
            return NULL;
    }
}
```

### 标识符

```c
CnType *check_identifier_type(CnSemScope *scope, CnAstExpr *expr) {
    CnSemSymbol *sym = cn_sem_scope_lookup(scope,
                                            expr->as.identifier.name,
                                            expr->as.identifier.name_length);
    if (!sym) {
        report_error("未定义的标识符");
        return NULL;
    }
    return sym->type;
}
```

### 二元运算

```c
CnType *check_binary_type(CnSemScope *scope, CnAstBinaryExpr *expr) {
    CnType *left_type = infer_expr_type(scope, expr->left);
    CnType *right_type = infer_expr_type(scope, expr->right);
    
    // 算术运算
    if (expr->op >= CN_AST_BINARY_ADD && expr->op <= CN_AST_BINARY_MOD) {
        if (!is_numeric_type(left_type)) {
            report_error("左操作数必须是数值类型");
        }
        if (!is_numeric_type(right_type)) {
            report_error("右操作数必须是数值类型");
        }
        // 返回较"大"的类型
        if (is_float_type(left_type) || is_float_type(right_type)) {
            return cn_type_new_primitive(CN_TYPE_FLOAT);
        }
        return cn_type_new_primitive(CN_TYPE_INT);
    }
    
    // 比较运算
    if (expr->op >= CN_AST_BINARY_EQ && expr->op <= CN_AST_BINARY_GE) {
        if (!cn_type_compatible(left_type, right_type)) {
            report_error("操作数类型不兼容");
        }
        return cn_type_new_primitive(CN_TYPE_BOOL);
    }
    
    // 逻辑运算
    if (expr->op >= CN_AST_BINARY_AND && expr->op <= CN_AST_BINARY_OR) {
        if (!is_bool_type(left_type) || !is_bool_type(right_type)) {
            report_error("逻辑运算需要布尔操作数");
        }
        return cn_type_new_primitive(CN_TYPE_BOOL);
    }
    
    return NULL;
}
```

### 赋值

```c
CnType *check_assign_type(CnSemScope *scope, CnAstAssignExpr *expr) {
    CnType *target_type = infer_expr_type(scope, expr->target);
    CnType *value_type = infer_expr_type(scope, expr->value);
    
    if (!cn_type_compatible(value_type, target_type)) {
        report_error("类型不兼容：无法将 '%T' 赋值给 '%T'",
                     value_type, target_type);
    }
    
    return target_type;
}
```

### 函数调用

```c
CnType *check_call_type(CnSemScope *scope, CnAstCallExpr *expr) {
    CnType *callee_type = infer_expr_type(scope, expr->callee);
    
    if (!is_function_type(callee_type)) {
        report_error("被调用对象不是函数");
        return NULL;
    }
    
    CnType *func_type = callee_type->as.function;
    
    // 检查参数数量
    if (expr->argument_count != func_type->param_count) {
        report_error("参数数量不匹配：期望 %zu 个，实际 %zu 个",
                     func_type->param_count, expr->argument_count);
        return func_type->return_type;
    }
    
    // 检查参数类型
    for (size_t i = 0; i < expr->argument_count; i++) {
        CnType *arg_type = infer_expr_type(scope, expr->arguments[i]);
        CnType *param_type = func_type->param_types[i];
        
        if (!cn_type_compatible(arg_type, param_type)) {
            report_error("参数 %zu 类型不兼容：期望 '%T'，实际 '%T'",
                         i + 1, param_type, arg_type);
        }
    }
    
    return func_type->return_type;
}
```

### 数组访问

```c
CnType *check_index_type(CnSemScope *scope, CnAstIndexExpr *expr) {
    CnType *array_type = infer_expr_type(scope, expr->array);
    CnType *index_type = infer_expr_type(scope, expr->index);
    
    if (!is_array_type(array_type) && !is_pointer_type(array_type)) {
        report_error("索引操作需要数组或指针类型");
        return NULL;
    }
    
    if (!is_int_type(index_type)) {
        report_error("索引必须是整数类型");
        return NULL;
    }
    
    if (is_array_type(array_type)) {
        return array_type->as.array.element_type;
    } else {
        return array_type->as.pointer_to;
    }
}
```

### 成员访问

```c
CnType *check_member_type(CnSemScope *scope, CnAstMemberExpr *expr) {
    CnType *object_type = infer_expr_type(scope, expr->object);
    
    if (is_struct_type(object_type)) {
        CnStructField *field = cn_type_struct_find_field(object_type,
                                                          expr->member_name,
                                                          expr->member_name_length);
        if (!field) {
            report_error("结构体没有字段 '%.*s'",
                         (int)expr->member_name_length,
                         expr->member_name);
            return NULL;
        }
        return field->type;
    }
    
    report_error("成员访问需要结构体类型");
    return NULL;
}
```

## 语句类型检查

### 变量声明

```c
void check_var_decl_type(CnSemScope *scope, CnAstVarDecl *decl) {
    CnType *declared_type = decl->declared_type;
    CnType *init_type = NULL;
    
    if (decl->initializer) {
        init_type = infer_expr_type(scope, decl->initializer);
        
        if (declared_type) {
            // 有显式类型声明
            if (!cn_type_compatible(init_type, declared_type)) {
                report_error("初始化类型不兼容：声明为 '%T'，实际为 '%T'",
                             declared_type, init_type);
            }
        } else {
            // 类型推断
            decl->inferred_type = init_type;
        }
    } else if (!declared_type) {
        report_error("变量声明必须有类型或初始化");
    }
}
```

### 返回语句

```c
void check_return_type(CnSemScope *scope, CnAstReturnStmt *ret, CnType *expected_type) {
    if (!ret->expr) {
        if (expected_type && expected_type->kind != CN_TYPE_VOID) {
            report_error("函数返回 '%T'，但返回语句没有值", expected_type);
        }
        return;
    }
    
    CnType *expr_type = infer_expr_type(scope, ret->expr);
    
    if (!cn_type_compatible(expr_type, expected_type)) {
        report_error("返回类型不兼容：期望 '%T'，实际为 '%T'",
                     expected_type, expr_type);
    }
}
```

### 条件语句

```c
void check_if_type(CnSemScope *scope, CnAstIfStmt *if_stmt) {
    CnType *cond_type = infer_expr_type(scope, if_stmt->condition);
    
    if (!is_bool_type(cond_type)) {
        report_error("条件必须是布尔类型，实际为 '%T'", cond_type);
    }
    
    check_block_type(scope, if_stmt->then_block);
    
    if (if_stmt->else_block) {
        check_block_type(scope, if_stmt->else_block);
    }
}
```

### 循环语句

```c
void check_while_type(CnSemScope *scope, CnAstWhileStmt *while_stmt) {
    CnType *cond_type = infer_expr_type(scope, while_stmt->condition);
    
    if (!is_bool_type(cond_type)) {
        report_error("循环条件必须是布尔类型");
    }
    
    check_block_type(scope, while_stmt->body);
}

void check_for_type(CnSemScope *scope, CnAstForStmt *for_stmt) {
    // 检查初始化语句
    check_stmt_type(scope, for_stmt->init);
    
    // 检查条件
    if (for_stmt->condition) {
        CnType *cond_type = infer_expr_type(scope, for_stmt->condition);
        if (!is_bool_type(cond_type)) {
            report_error("循环条件必须是布尔类型");
        }
    }
    
    // 检查更新
    if (for_stmt->update) {
        infer_expr_type(scope, for_stmt->update);
    }
    
    // 检查循环体
    check_block_type(scope, for_stmt->body);
}
```

## 类型错误报告

### 常见类型错误

| 错误代码 | 描述 | 示例 |
|----------|------|------|
| `SEM_TYPE_MISMATCH` | 类型不匹配 | `整数 x = "文本";` |
| `SEM_INVALID_OPERATOR` | 无效运算符 | `"文本" + 10;` |
| `SEM_WRONG_ARGUMENTS` | 参数错误 | `加法 (5, "文本");` |
| `SEM_RETURN_TYPE_MISMATCH` | 返回类型不匹配 | `返回 "文本";` (期望整数) |
| `SEM_UNDEFINED_FIELD` | 未定义字段 | `p.未知字段;` |
| `SEM_NOT_ASSIGNABLE` | 不可赋值 | `10 = x;` |

### 错误报告函数

```c
void report_type_error(CnDiagnostics *diagnostics,
                       CnDiagCode code,
                       const char *message,
                       const char *filename,
                       size_t line,
                       size_t column,
                       ...) {
    va_list args;
    va_start(args, column);
    
    cn_support_diagnostics_report_v(diagnostics,
                                    CN_DIAG_SEVERITY_ERROR,
                                    code,
                                    filename,
                                    line,
                                    column,
                                    message,
                                    args);
    
    va_end(args);
}
```

## 使用示例

### 类型检查示例

```cn
// 正确的类型使用
函数 正确示例 () {
    整数 a = 10;
    整数 b = 20;
    整数 和 = a + b;
    
    如果 (和 > 25) {
        打印 ("和大");
    }
    
    返回 和;
}

// 类型错误示例
函数 错误示例 () {
    整数 x = "文本";  // 错误：类型不匹配
    整数 y = x + "文本";  // 错误：无效运算符
    返回 "文本";  // 错误：返回类型不匹配
}
```
