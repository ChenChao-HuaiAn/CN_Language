# 类型系统

## 概述

CN_Language 的类型系统支持静态类型检查、类型推断和类型转换。类型系统确保程序中的类型使用是正确的。

## 文件位置

[`src/semantics/symbols/type_system.c`](../../src/semantics/symbols/type_system.c:1)

## 类型数据结构

### CnType 结构

```c
typedef struct CnType {
    CnTypeKind kind;             // 类型种类
    union {
        // 指针类型
        CnType *pointer_to;
        
        // 数组类型
        struct {
            CnType *element_type;
            size_t length;
        } array;
        
        // 函数类型
        struct {
            CnType *return_type;
            CnType **param_types;
            size_t param_count;
        } function;
        
        // 结构体类型
        struct {
            const char *name;
            size_t name_length;
            CnStructField *fields;
            size_t field_count;
            CnSemScope *decl_scope;
        } struct_type;
        
        // 枚举类型
        struct {
            const char *name;
            size_t name_length;
            CnSemScope *enum_scope;
        } enum_type;
    } as;
} CnType;
```

### 类型种类 (CnTypeKind)

```c
typedef enum {
    // 基础类型
    CN_TYPE_VOID,
    CN_TYPE_INT,
    CN_TYPE_FLOAT,
    CN_TYPE_BOOL,
    CN_TYPE_CHAR,
    CN_TYPE_STRING,
    
    // 定点类型
    CN_TYPE_INT8,
    CN_TYPE_INT16,
    CN_TYPE_INT32,
    CN_TYPE_INT64,
    CN_TYPE_UINT8,
    CN_TYPE_UINT16,
    CN_TYPE_UINT32,
    CN_TYPE_UINT64,
    CN_TYPE_FLOAT32,
    CN_TYPE_FLOAT64,
    
    // 复合类型
    CN_TYPE_POINTER,
    CN_TYPE_ARRAY,
    CN_TYPE_FUNCTION,
    CN_TYPE_STRUCT,
    CN_TYPE_ENUM,
    
    // 特殊类型
    CN_TYPE_MEMORY_ADDRESS,
    CN_TYPE_UNKNOWN,
} CnTypeKind;
```

## 类型创建

### 基础类型

```c
// 创建基础类型
CnType *cn_type_new_primitive(CnTypeKind kind);

// 示例
CnType *int_type = cn_type_new_primitive(CN_TYPE_INT);
CnType *float_type = cn_type_new_primitive(CN_TYPE_FLOAT);
CnType *bool_type = cn_type_new_primitive(CN_TYPE_BOOL);
CnType *void_type = cn_type_new_primitive(CN_TYPE_VOID);
```

### 指针类型

```c
CnType *cn_type_new_pointer(CnType *base);

// 示例
CnType *int_ptr_type = cn_type_new_pointer(int_type);  // 整数*
CnType *void_ptr_type = cn_type_new_pointer(void_type); // 空*
```

### 数组类型

```c
CnType *cn_type_new_array(CnType *element, size_t length);

// 示例
CnType *int_array_type = cn_type_new_array(int_type, 10);  // 整数 [10]
CnType *unbound_array = cn_type_new_array(int_type, 0);     // 整数 []
```

### 函数类型

```c
CnType *cn_type_new_function(CnType *return_type, 
                             CnType **param_types, 
                             size_t param_count);

// 示例：函数 加法 (整数，整数) 返回 整数
CnType *param_types[2] = {int_type, int_type};
CnType *func_type = cn_type_new_function(int_type, param_types, 2);
```

### 结构体类型

```c
CnType *cn_type_new_struct(const char *name, size_t name_length,
                           CnStructField *fields, size_t field_count,
                           CnSemScope *decl_scope);

// 示例
结构体 点 {
    整数 x;
    整数 y;
}

CnStructField fields[2] = {
    {.name = "x", .type = int_type},
    {.name = "y", .type = int_type}
};
CnType *point_type = cn_type_new_struct("点", 3, fields, 2, NULL);
```

### 枚举类型

```c
CnType *cn_type_new_enum(const char *name, size_t name_length);

// 示例
枚举 颜色 {
    红，
    绿，
    蓝
}

CnType *color_type = cn_type_new_enum("颜色", 5);
```

## 类型比较

### 类型相等

```c
bool cn_type_equals(CnType *a, CnType *b);
```

**比较规则：**
1. 基础类型：种类相同即相等
2. 指针类型：指向的类型相等
3. 数组类型：元素类型和长度都相等
4. 函数类型：返回类型和参数类型都相等
5. 结构体/枚举：名称相等

**示例：**
```c
CnType *int1 = cn_type_new_primitive(CN_TYPE_INT);
CnType *int2 = cn_type_new_primitive(CN_TYPE_INT);

if (cn_type_equals(int1, int2)) {
    printf("类型相等\n");
}
```

### 类型兼容

```c
bool cn_type_compatible(CnType *a, CnType *b);
```

**兼容规则：**
1. 类型相等即兼容
2. 整数类型之间兼容
3. 浮点类型之间兼容
4. 整数可以转换为浮点
5. 数组可以退化为指针

**示例：**
```c
// 整数类型之间兼容
CnType *int32 = cn_type_new_primitive(CN_TYPE_INT32);
CnType *int64 = cn_type_new_primitive(CN_TYPE_INT64);
cn_type_compatible(int32, int64);  // true

// 数组到指针
CnType *array_type = cn_type_new_array(int_type, 10);
CnType *ptr_type = cn_type_new_pointer(int_type);
cn_type_compatible(array_type, ptr_type);  // true
```

## 类型推断

### 表达式类型推断

```c
CnType *infer_expr_type(CnSemScope *scope, CnAstExpr *expr, CnDiagnostics *diagnostics);
```

**推断规则：**

| 表达式类型 | 推断结果 |
|------------|----------|
| 整数字面量 | `整数` |
| 浮点字面量 | `小数` |
| 字符串字面量 | `字符串` |
| 布尔字面量 | `布尔` |
| 标识符 | 符号的类型 |
| 二元运算 | 根据操作数推断 |
| 函数调用 | 函数返回类型 |
| 赋值 | 右值类型 |

### 函数返回类型推断

```c
CnType *infer_function_return_type(CnSemScope *scope, 
                                    CnAstBlockStmt *block, 
                                    CnDiagnostics *diagnostics);
```

**示例：**
```cn
// 返回类型可推断
函数 加法 (a, b) {  // 参数和返回类型都可推断
    返回 a + b;  // 推断返回 整数
}
```

## 类型转换

### 隐式转换

```c
// 隐式转换规则
// 1. 整数提升
// 2. 整数到浮点
// 3. 子类型到父类型
// 4. 数组到指针
```

**示例：**
```cn
整数 i = 10;
小数 f = i;  // 整数自动转换为小数

整数 数组 [10];
整数* 指针 = 数组;  // 数组退化为指针
```

### 显式转换

```cn
// 显式类型转换语法
整数 i = 10;
小数 f = (小数) i;  // 显式转换

空* 空指针 = (空*) 整数指针;
```

```c
CnStructField *field = cn_type_struct_find_field(point_type, "x", 1);
```

### 字段访问检查

```cn
结构体 点 {
    整数 x;
    整数 y;
}
```

## 函数类型检查

### 参数类型检查

```c
void check_function_call(CnSemScope *scope,
                         CnAstCallExpr *call,
                         CnDiagnostics *diagnostics);
```

**检查内容：**
1. 参数数量匹配
2. 参数类型兼容
3. 返回类型使用正确

**示例：**
```cn
函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

整数 结果 = 加法 (5, 3);      // 正确
整数 错误 = 加法 (5);         // 错误：参数数量不匹配
整数 错误 2 = 加法 (5, "文本"); // 错误：类型不匹配
```

## 类型错误

### 常见类型错误

| 错误 | 描述 | 示例 |
|------|------|------|
| 类型不匹配 | 赋值类型不兼容 | `整数 x = "文本";` |
| 未定义类型 | 使用了未定义的类型 | `未知类型 x;` |
| 字段不存在 | 访问不存在的字段 | `p.z = 10;` |
| 参数不匹配 | 函数调用参数错误 | `加法 (5, "文本");` |
| 返回类型错误 | 返回类型不匹配 | `返回 "文本";` (期望整数) |

### 错误报告

```c
                       const char *message,
                       const char *filename,
                       size_t line,
                       size_t column);
```

## 使用示例

### 完整的类型检查示例

```c
#include "cnlang/frontend/semantics.h"
#include <stdio.h>

int main() {
    // 创建类型
    CnType *int_type = cn_type_new_primitive(CN_TYPE_INT);
    CnType *float_type = cn_type_new_primitive(CN_TYPE_FLOAT);
    
    // 创建指针类型
    CnType *int_ptr = cn_type_new_pointer(int_type);
    
    // 创建数组类型
    CnType *int_array = cn_type_new_array(int_type, 10);
    
    // 创建函数类型
    CnType *params[2] = {int_type, int_type};
    CnType *func_type = cn_type_new_function(int_type, params, 2);
    
    // 类型比较
    if (cn_type_equals(int_type, int_type)) {
        printf("整数类型相等\n");
    }
    
    // 类型兼容检查
    if (cn_type_compatible(int_type, float_type)) {
        printf("整数和浮点兼容\n");
    }
    
    // 清理
    free(int_type);
    free(float_type);
    free(int_ptr);
    free(int_array);
    free(func_type);
    
    return 0;
}
```
