# 名称解析

## 概述

名称解析是将源代码中的标识符绑定到相应声明的过程。名称解析在作用域树中查找标识符，并报告未定义或歧义的错误。

## 文件位置

[`src/semantics/checker/semantic_passes.c`](../../src/semantics/checker/semantic_passes.c:1)

## 解析流程

```
AST → [构建作用域] → [解析名称] → [类型检查]
         ↓              ↓
    符号表填充    标识符绑定
```

## 名称解析 Pass

### 入口函数

```c
bool cn_sem_resolve_names(CnSemScope *global_scope,
                          CnAstProgram *program,
                          CnDiagnostics *diagnostics);
```

**功能：**
1. 处理全局函数
2. 处理模块内函数
3. 解析所有块中的名称

### 块名称解析

```c
static void resolve_block_names(CnSemScope *scope, 
                                CnAstBlockStmt *block, 
                                CnDiagnostics *diagnostics);
```

**处理内容：**
- 遍历块中所有语句
- 为每个语句解析名称
- 创建嵌套块作用域

### 语句名称解析

```c
static void resolve_stmt_names(CnSemScope *scope, 
                               CnAstStmt *stmt, 
                               CnDiagnostics *diagnostics);
```

**处理不同语句类型：**

| 语句类型 | 解析行为 |
|----------|----------|
| 代码块 | 创建新作用域，递归解析 |
| 变量声明 | 解析初始化表达式，插入符号 |
| 表达式语句 | 解析表达式中的名称 |
| 返回语句 | 解析返回表达式 |
| 条件语句 | 解析条件和分支块 |
| 循环语句 | 解析条件和循环体 |
| For 循环 | 创建循环作用域，解析各部分 |

### 表达式名称解析

```c
static void resolve_expr_names(CnSemScope *scope, 
                               CnAstExpr *expr, 
                               CnDiagnostics *diagnostics);
```

**处理不同表达式类型：**

| 表达式类型 | 解析行为 |
|------------|----------|
| 标识符 | 在作用域中查找符号 |
| 二元运算 | 递归解析左右操作数 |
| 函数调用 | 解析被调用函数和参数 |
| 赋值 | 解析目标和值 |
| 逻辑运算 | 递归解析操作数 |
| 一元运算 | 递归解析操作数 |
| 数组字面量 | 解析所有元素 |
| 索引访问 | 解析数组和索引 |
| 成员访问 | 解析对象和成员 |

## 作用域构建

### 全局作用域

```c
// 创建全局作用域
CnSemScope *global_scope = cn_sem_scope_new(CN_SEM_SCOPE_GLOBAL, NULL);

// 插入内置符号
insert_builtin_symbols(global_scope);

// 处理顶级声明
for (each decl in program) {
    if (decl is function) {
        CnSemSymbol *sym = cn_sem_scope_insert_symbol(global_scope, 
                                                       decl->name,
                                                       CN_SEM_SYMBOL_FUNCTION);
        sym->as.function.decl = decl;
    }
}
```

### 函数作用域

```c
// 为函数创建作用域
CnSemScope *func_scope = cn_sem_scope_new(CN_SEM_SCOPE_FUNCTION, global_scope);
cn_sem_scope_set_name(func_scope, func_decl->name, func_decl->name_length);

// 插入参数符号
for (size_t i = 0; i < func_decl->param_count; i++) {
    CnSemSymbol *sym = cn_sem_scope_insert_symbol(func_scope,
                                                   func_decl->params[i].name,
                                                   func_decl->params[i].name_length,
                                                   CN_SEM_SYMBOL_VARIABLE);
    if (sym) {
        sym->is_const = func_decl->params[i].is_const;
    }
}
```

### 块作用域

```c
// 为代码块创建作用域
CnSemScope *block_scope = cn_sem_scope_new(CN_SEM_SCOPE_BLOCK, parent_scope);

// 处理块中的变量声明
for (each stmt in block) {
    if (stmt is var_decl) {
        cn_sem_scope_insert_symbol(block_scope, 
                                    stmt->as.var_decl.name,
                                    stmt->as.var_decl.name_length,
                                    CN_SEM_SYMBOL_VARIABLE);
    }
}

// 递归处理嵌套块
for (each stmt in block) {
    if (stmt is block) {
        resolve_block_names(block_scope, stmt->as.block, diagnostics);
    }
}

// 释放块作用域（符号已绑定到 AST）
cn_sem_scope_free(block_scope);
```

### For 循环作用域

```c
// For 循环创建独立作用域
CnSemScope *for_scope = cn_sem_scope_new(CN_SEM_SCOPE_BLOCK, scope);

// 解析初始化语句（可能声明变量）
resolve_stmt_names(for_scope, stmt->as.for_stmt.init, diagnostics);

// 解析条件表达式
resolve_expr_names(for_scope, stmt->as.for_stmt.condition, diagnostics);

// 解析更新表达式
resolve_expr_names(for_scope, stmt->as.for_stmt.update, diagnostics);

// 解析循环体
resolve_block_names(for_scope, stmt->as.for_stmt.body, diagnostics);

// 释放作用域
cn_sem_scope_free(for_scope);
```

## 符号查找

### 标识符解析

```c
// 解析标识符表达式
case CN_AST_EXPR_IDENTIFIER: {
    CnSemSymbol *sym = cn_sem_scope_lookup(scope, 
                                            expr->as.identifier.name,
                                            expr->as.identifier.name_length);
    if (!sym) {
        cn_support_diag_semantic_error_undefined_identifier(
            diagnostics, NULL, 0, 0, expr->as.identifier.name);
    }
    break;
}
```

### 成员访问解析

```c
// 解析成员访问
case CN_AST_EXPR_MEMBER_ACCESS: {
    // 先解析对象
    resolve_expr_names(scope, expr->as.member.object, diagnostics);
    
    // 如果对象是标识符，查找其符号
    if (expr->as.member.object->kind == CN_AST_EXPR_IDENTIFIER) {
        CnSemSymbol *obj_sym = cn_sem_scope_lookup(
            scope,
            expr->as.member.object->as.identifier.name,
            expr->as.member.object->as.identifier.name_length);
        
        if (obj_sym) {
            // 根据符号类型验证成员
            if (obj_sym->kind == CN_SEM_SYMBOL_MODULE) {
                // 验证模块成员
                verify_module_member(obj_sym, expr->as.member.member_name);
            } else if (obj_sym->kind == CN_SEM_SYMBOL_STRUCT) {
                // 验证结构体字段
                verify_struct_field(obj_sym, expr->as.member.member_name);
            }
        }
    }
    break;
}
```

## 错误处理

### 未定义标识符

```c
if (!sym) {
    cn_support_diagnostics_report(diagnostics,
                                  CN_DIAG_SEVERITY_ERROR,
                                  CN_DIAG_CODE_SEM_UNDEFINED_IDENTIFIER,
                                  filename,
                                  line,
                                  column,
                                  "未定义的标识符：%.*s",
                                  (int)name_length,
                                  name);
}
```

**示例：**
```cn
函数 测试 () {
    打印 (未定义的变量);  // 错误：未定义的标识符
}
```

### 重复定义

```c
CnSemSymbol *existing = cn_sem_scope_lookup_shallow(scope, name, name_length);
if (existing) {
    cn_support_diagnostics_report(diagnostics,
                                  CN_DIAG_SEVERITY_ERROR,
                                  CN_DIAG_CODE_SEM_DUPLICATE_SYMBOL,
                                  filename,
                                  line,
                                  column,
                                  "符号 '%.*s' 重复定义",
                                  (int)name_length,
                                  name);
}
```

**示例：**
```cn
函数 测试 () {
    整数 x = 10;
    整数 x = 20;  // 错误：符号 'x' 重复定义
}
```

### 作用域错误

```cn
// 变量超出作用域
函数 外层 () {
    如果 (真) {
        整数 内层变量 = 10;
    }
    打印 (内层变量);  // 错误：未定义的标识符
}

// 遮蔽警告
整数 x = 10;
函数 测试 () {
    整数 x = 20;  // 警告：遮蔽了外层变量 'x'
}
```

## 模块名称解析

### 模块路径解析

```c
// 解析点分模块路径
从 图形。二维 导入 点;

// 解析过程：
// 1. 查找 '图形' 模块
// 2. 在图形模块中查找 '二维' 子模块
// 3. 在二维模块中查找 '点' 符号
```

### 导入语句解析

```c
// 处理导入语句
case CN_AST_STMT_IMPORT: {
    CnAstImportStmt *import = &stmt->as.import_stmt;
    
    // 解析模块路径
    CnModule *module = resolve_module_path(import->module_path);
    
    if (!module) {
        report_error("模块未找到：%.*s", 
                     (int)import->module_path_length,
                     import->module_path);
        break;
    }
    
    // 解析导入的成员
    for (size_t i = 0; i < import->member_count; i++) {
        CnSemSymbol *sym = cn_sem_scope_lookup(module->scope,
                                                import->members[i].name,
                                                import->members[i].name_length);
        if (!sym) {
            report_error("模块 '%.*s' 中没有成员 '%.*s'",
                         (int)import->module_path_length,
                         import->module_path,
                         (int)import->members[i].name_length,
                         import->members[i].name);
        }
        
        // 将导入的符号插入当前作用域
        const char *local_name = import->members[i].alias 
                                  ? import->members[i].alias 
                                  : import->members[i].name;
        cn_sem_scope_insert_symbol(scope, 
                                    local_name,
                                    strlen(local_name),
                                    sym->kind);
    }
    break;
}
```

## 使用示例

### 完整的名称解析示例

```c
#include "cnlang/frontend/semantics.h"
#include <stdio.h>

int main() {
    CnDiagnostics diagnostics;
    cn_support_diagnostics_init(&diagnostics);
    
    // 创建全局作用域
    CnSemScope *global = cn_sem_scope_new(CN_SEM_SCOPE_GLOBAL, NULL);
    
    // 插入全局符号
    cn_sem_scope_insert_symbol(global, "全局变量", 8, CN_SEM_SYMBOL_VARIABLE);
    
    // 创建函数作用域
    CnSemScope *func_scope = cn_sem_scope_new(CN_SEM_SCOPE_FUNCTION, global);
    cn_sem_scope_set_name(func_scope, "主程序", 7);
    
    // 插入参数
    cn_sem_scope_insert_symbol(func_scope, "参数", 4, CN_SEM_SYMBOL_VARIABLE);
    
    // 模拟解析 AST 中的标识符
    const char *identifiers[] = {"全局变量", "参数", "未定义符号"};
    
    for (int i = 0; i < 3; i++) {
        CnSemSymbol *sym = cn_sem_scope_lookup(func_scope, 
                                                identifiers[i],
                                                strlen(identifiers[i]));
        if (sym) {
            printf("找到符号：%.*s\n", (int)sym->name_length, sym->name);
        } else {
            printf("未找到符号：%s\n", identifiers[i]);
        }
    }
    
    // 清理
    cn_sem_scope_free(func_scope);
    cn_sem_scope_free(global);
    
    return 0;
}
```
