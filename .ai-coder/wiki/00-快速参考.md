# CN_Language 快速参考

## 语言速查

### 基础类型

```cn
整数 x = 10;          // 整数
小数 y = 3.14;        // 浮点数
布尔 b = 真;          // 布尔
字符串 s = "你好";    // 字符串
字符 c = 'A';         // 字符
空 结果;              // 空类型
```

### 控制流

```cn
// 条件
如果 (x > 0) {
    打印 ("正数");
} 否则 如果 (x < 0) {
    打印 ("负数");
} 否则 {
    打印 ("零");
}

// 选择
选择 (选项) {
    情况 1: 打印 ("一"); 中断;
    情况 2: 打印 ("二"); 中断;
    默认：打印 ("其他");
}

// 循环
当 (条件) {
    // ...
}

循环 (整数 i = 0; i < 10; i = i + 1) {
    // ...
}
```

### 函数

```cn
// 定义
函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

// 调用
整数 结果 = 加法 (5, 3);

// 中断处理程序
中断处理程序 定时器处理 (整数 中断号) {
    // ...
}
```

### 复合类型

```cn
// 结构体
结构体 点 {
    整数 x;
    整数 y;
}

点 p = 点 {x: 10, y: 20};
打印 (p.x);

// 枚举
枚举 颜色 {
    红，
    绿，
    蓝
}

颜色 c = 颜色。红;

// 数组
整数 数组 [5] 数字 = {1, 2, 3, 4, 5};
数字 [0] = 10;
```

### 指针

```cn
整数 x = 10;
整数* 指针 = &x;
打印 (*指针);

// 函数指针
函数 类型 (*操作指针) (整数，整数) = 加法;
整数 结果 = 操作指针 (5, 3);
```

### 模块

```cn
// 定义模块
模块 数学模块;

导出 函数 加法 (整数 a, 整数 b) {
    返回 a + b;
}

// 导入
导入 数学模块;
整数 和 = 数学模块。加法 (5, 3);

// 选择性导入
从 数学模块 导入 加法，减法;
整数 结果 = 加法 (10, 20);
```

### 预处理器


## 工具速查

### 编译器 (cnc)

```bash
# 基本编译
cnc 程序.cn -o 程序

# 优化编译
cnc 程序.cn -O2 -o 程序

# 生成 C 代码
cnc 程序.cn --emit-c -o 程序.c

# Freestanding 模式
cnc 内核.cn --freestanding -o 内核.bin
```

### 格式化 (cnfmt)

```bash
# 格式化文件
cnfmt 文件.cn

# 检查格式
cnfmt --check 文件.cn

# 递归格式化
cnfmt -r 目录/
```

### 静态检查 (cncheck)

```bash
# 检查文件
cncheck 文件.cn

# 忽略特定警告
cncheck --ignore=unused-variable 文件.cn

# JSON 输出
cncheck --json 文件.cn
```

### 交互式解释器 (cnrepl)

```bash
# 启动 REPL
cnrepl

# 加载文件
cnrepl 初始化.cn

# 特殊命令
:help      # 帮助
:quit      # 退出
:load 文件  # 加载文件
:history   # 历史
```

### 性能分析 (cnperf)

```bash
# 分析程序
cnperf 程序.cn

# 热点分析
cnperf --hotspot 10 程序.cn

# 调用图
cnperf --call-graph 程序.cn
```

## 构建速查

### CMake 配置

```bash
# 调试版本
cmake -B build -DCMAKE_BUILD_TYPE=Debug

# 发布版本
cmake -B build -DCMAKE_BUILD_TYPE=Release

# 带覆盖率
cmake -B build -DCMAKE_C_FLAGS="--coverage"
```

### 编译和测试

```bash
# 编译
cmake --build build

# 运行测试
cd build && ctest --output-on-failure

# 安装
sudo cmake --install build
```

### 调试

```bash
# GDB 调试
gdb ./build/src/cnc

# Valgrind
valgrind --leak-check=full ./build/src/cnc 程序.cn

# AddressSanitizer
cmake -B build -DCMAKE_C_FLAGS="-fsanitize=address"
```

## 常见错误

### 类型错误

```
错误：类型不兼容：无法将 '字符串' 赋值给 '整数'
解决：检查变量类型和赋值类型
```

### 未定义符号

```
错误：未定义的标识符 '未定义变量'
解决：检查变量是否已声明，作用域是否正确
```

### 模块未找到

```
错误：模块未找到：数学模块
解决：检查模块路径和搜索路径配置
```

### 内存泄漏

```
Valgrind: 100 bytes in 1 blocks are definitely lost
解决：检查所有 malloc 是否有对应的 free
```
