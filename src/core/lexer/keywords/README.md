# 关键字识别模块

## 概述

关键字识别模块负责识别CN_Language的70个中文关键字。它提供高效的关键字查找和识别功能，支持将标识符词素映射到相应的令牌类型。

## 文件说明

### CN_lexer_keywords.h
- 关键字识别函数声明
- 关键字表定义
- 公共接口函数

### CN_lexer_keywords.c
- 关键字识别实现
- 关键字表初始化
- 查找算法实现

## 功能特性

### 1. 关键字支持
- **70个中文关键字** - 完整的CN_Language关键字集
- **9个类别** - 按功能分类的关键字
- **快速查找** - 高效的关键字识别算法

### 2. 关键字类别
- **变量和类型声明**：变量、常量、类型、类、对象等
- **控制结构**：如果、否则、当、循环、中断、继续等
- **函数相关**：函数、返回、主程序、无等
- **逻辑运算符**：与、或、非等
- **字面量**：真、假、空等
- **模块系统**：模块、导入、导出等
- **运算符关键字**：加、减、乘、除等
- **类型关键字**：整数、浮点数、字符串、布尔等
- **其他关键字**：尝试、捕获、抛出等

### 3. 识别算法
- **哈希查找** - 快速的关键字查找
- **精确匹配** - 确保准确的关键字识别
- **大小写敏感** - 中文关键字大小写敏感

## 主要函数

### 识别函数
- `F_identify_keyword()` - 识别关键字类型
- `F_is_keyword()` - 检查词素是否为关键字

### 辅助函数
- `F_initialize_keyword_table()` - 初始化关键字表
- `F_destroy_keyword_table()` - 清理关键字表

## 使用示例

```c
#include "CN_lexer_keywords.h"

// 识别关键字
const char* lexeme = "变量";
Eum_TokenType type = F_identify_keyword(lexeme);

if (type != Eum_TOKEN_IDENTIFIER) {
    printf("识别到关键字: %s -> %d\n", lexeme, type);
}

// 检查是否为关键字
bool is_keyword = F_is_keyword(lexeme);
if (is_keyword) {
    printf("%s 是关键字\n", lexeme);
}
```

## 关键字表设计

### 数据结构
- **静态数组** - 编译时初始化的关键字表
- **哈希表** - 运行时构建的快速查找表
- **双向映射** - 关键字到令牌类型的映射

### 查找算法
1. **长度过滤** - 首先检查词素长度
2. **哈希查找** - 使用哈希值快速定位
3. **精确匹配** - 验证字符串完全匹配
4. **类型返回** - 返回对应的令牌类型
