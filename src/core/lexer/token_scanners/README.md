# 令牌扫描器模块

## 概述

令牌扫描器模块负责识别和提取各种类型的令牌，包括标识符、数字字面量、字符串字面量等。它是词法分析器的核心组件，将字符流转换为有意义的令牌序列。

## 文件说明

### CN_lexer_token_scanners.h
- 令牌扫描函数声明
- 扫描器状态类型定义
- 公共接口函数

### CN_lexer_token_scanners.c
- 各种令牌类型的扫描实现
- 令牌识别逻辑
- 错误处理

## 功能特性

### 1. 令牌类型支持
- **标识符扫描** (`F_scan_identifier`) - 扫描标识符和关键字
- **数字扫描** (`F_scan_number`) - 扫描整数和浮点数
- **字符串扫描** (`F_scan_string`) - 扫描字符串字面量
- **主扫描函数** (`F_scan_next_token`) - 根据字符类型选择适当的扫描器

### 2. 扫描算法
- **前瞻字符分析** - 根据下一个字符选择扫描器
- **词素提取** - 提取令牌的文本表示
- **位置跟踪** - 记录令牌的起始位置
- **错误恢复** - 在错误情况下继续扫描

### 3. 字符分类
- **字母字符** - 标识符的起始字符
- **数字字符** - 数字字面量的组成部分
- **运算符字符** - 运算符的组成部分
- **分隔符字符** - 分隔符识别

## 主要函数

### 扫描函数
- `F_scan_next_token()` - 扫描下一个令牌（主函数）
- `F_scan_identifier()` - 扫描标识符或关键字
- `F_scan_number()` - 扫描数字字面量
- `F_scan_string()` - 扫描字符串字面量

### 辅助函数
- `F_is_alpha()` - 检查字符是否为字母
- `F_is_digit()` - 检查字符是否为数字
- `F_is_alpha_numeric()` - 检查字符是否为字母或数字

## 使用示例

```c
#include "CN_lexer_token_scanners.h"

// 假设已有扫描器状态
Stru_LexerScannerState_t* state = ...;

// 扫描下一个令牌
Stru_Token_t* token = F_scan_next_token(state);

if (token != NULL) {
    // 处理令牌...
    F_destroy_token(token);
}
```

## 扫描流程

1. **跳过空白字符** - 调用扫描器的空白跳过函数
2. **检查文件结束** - 如果到达文件末尾，返回EOF令牌
3. **字符分类** - 根据下一个字符选择扫描器
4. **令牌提取** - 调用相应的扫描函数
5. **错误处理** - 处理扫描过程中的错误
