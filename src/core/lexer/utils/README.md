# 工具函数模块

## 概述

工具函数模块提供词法分析器所需的各种辅助函数，包括字符分类、字符串处理、内存管理等。它是词法分析器的基础工具库，为其他模块提供通用的功能支持。

## 文件说明

### CN_lexer_utils.h
- 工具函数声明
- 字符分类函数
- 字符串处理函数
- 内存管理函数

### CN_lexer_utils.c
- 工具函数实现
- 字符分类逻辑
- 字符串处理实现
- 内存管理实现

## 功能特性

### 1. 字符分类
- **字母检查** (`F_is_alpha`) - 检查字符是否为字母（包括中文字符）
- **数字检查** (`F_is_digit`) - 检查字符是否为数字
- **字母数字检查** (`F_is_alpha_numeric`) - 检查字符是否为字母或数字
- **空白字符检查** (`F_is_whitespace`) - 检查字符是否为空白字符
- **运算符字符检查** (`F_is_operator_char`) - 检查字符是否为运算符字符
- **分隔符字符检查** (`F_is_delimiter_char`) - 检查字符是否为分隔符字符

### 2. 字符串处理
- **字符串复制** (`F_copy_string`) - 安全地复制字符串
- **字符串比较** (`F_compare_strings`) - 比较两个字符串
- **字符串长度** (`F_string_length`) - 获取字符串长度
- **字符串连接** (`F_concat_strings`) - 连接两个字符串

### 3. 内存管理
- **安全分配** (`F_safe_malloc`) - 带错误检查的内存分配
- **安全释放** (`F_safe_free`) - 安全的释放内存
- **内存复制** (`F_safe_memcpy`) - 带边界检查的内存复制

## 使用示例

```c
#include "CN_lexer_utils.h"

// 字符分类示例
char c = 'A';
if (F_is_alpha(c)) {
    printf("%c 是字母\n", c);
}

if (F_is_digit('5')) {
    printf("5 是数字\n");
}

// 字符串处理示例
const char* src = "Hello";
char* dest = F_copy_string(src);
if (dest != NULL) {
    printf("复制的字符串: %s\n", dest);
    F_safe_free(dest);
}

// 内存管理示例
int* array = (int*)F_safe_malloc(10 * sizeof(int));
if (array != NULL) {
    // 使用数组...
    F_safe_free(array);
}
```
