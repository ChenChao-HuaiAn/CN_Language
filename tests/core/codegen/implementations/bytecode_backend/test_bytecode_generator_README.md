# 字节码生成器测试模块

## 概述

本目录包含CN_Language项目字节码生成器的测试代码。字节码生成器是字节码后端的重要组成部分，负责从AST和IR生成字节码程序。

## 目录结构

```
tests/core/codegen/implementations/bytecode_backend/
├── test_bytecode_generator.c      # 字节码生成器测试主文件
└── test_bytecode_generator_README.md  # 本文件
```

## 构建和运行

### 独立运行
```bash
cd tests/core/codegen/implementations/bytecode_backend
gcc -o test_bytecode_generator test_bytecode_generator.c -I../../../../../../src -I../../../../../../src/core/codegen
./test_bytecode_generator
```

### 集成运行
通过总测试运行器运行：
```bash
cd tests/core/codegen
gcc -o test_codegen_all test_codegen_all.c -I../../../../src -I../../../../src/core/codegen
./test_codegen_all
```

### 使用Makefile
```bash
cd tests
make test_codegen
```
