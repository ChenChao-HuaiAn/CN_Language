# 字节码后端测试模块

## 概述

本目录包含CN_Language项目字节码后端的测试代码。字节码后端是代码生成器的重要组成部分，负责将CN_Language的抽象语法树或中间代码转换为字节码程序，供虚拟机解释执行。

## 目录结构

```
tests/core/codegen/implementations/bytecode_backend/
├── README.md                    # 本文件
├── test_bytecode_backend.c      # 字节码后端测试主文件
└── (未来可能添加更多测试文件)
```

## 构建和运行

### 独立运行
```bash
cd tests/core/codegen/implementations/bytecode_backend
gcc -o test_bytecode_backend test_bytecode_backend.c -I../../../../../../src -I../../../../../../src/core/codegen
./test_bytecode_backend
```

### 集成运行
通过总测试运行器运行：
```bash
cd tests/core/codegen
gcc -o test_codegen_all test_codegen_all.c -I../../../../src -I../../../../src/core/codegen
./test_codegen_all
```

### 使用Makefile
```bash
cd tests
make test_codegen
```
