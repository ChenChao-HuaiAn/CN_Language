# 基础优化器测试模块

## 概述

本目录包含CN_Language项目基础优化器的测试代码。基础优化器是代码生成器的重要组成部分，提供常量折叠、死代码消除、公共子表达式消除、强度削减和窥孔优化等基础优化功能。

## 目录结构

```
tests/core/codegen/optimizers/basic_optimizer/
├── README.md                    # 本文件
├── test_basic_optimizer.c       # 基础优化器测试主文件
└── (未来可能添加更多测试文件)
```

- 验证特定优化类型支持
- 测试无效优化类型处理

- 测试优化器插件信息获取
- 验证信息字段完整性
- 检查版本号正确性

### 5. 优化器初始化测试
- 测试优化器初始化功能
- 验证重复初始化处理

## 构建和运行

### 独立运行
```bash
cd tests/core/codegen/optimizers/basic_optimizer
gcc -o test_basic_optimizer test_basic_optimizer.c -I../../../../../../src -I../../../../../../src/core/codegen
./test_basic_optimizer
```

### 集成运行
通过总测试运行器运行：
```bash
cd tests/core/codegen
gcc -o test_codegen_all test_codegen_all.c -I../../../../src -I../../../../src/core/codegen
./test_codegen_all
```

### 使用Makefile
```bash
cd tests
make test_codegen
```
