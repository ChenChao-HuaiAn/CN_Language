#### 实现的功能

1. **枚举成员自动添加前缀**
   - 生成的C代码中，枚举成员自动添加枚举类型名作为前缀
   - 例如：`enum 颜色 { 颜色_红, 颜色_绿, 颜色_蓝 }`

2. **枚举成员访问语义分析增强**
   - 支持直接成员名查找（如 `错误`）
   - 支持带枚举类型名前缀的成员名查找（如 `诊断严重级别_错误`）
   - 支持遍历所有成员，查找以 `_成员名` 结尾的成员（如 `诊断_错误` 匹配 `错误`）

#### 解决的问题

在C语言中，枚举成员在全局命名空间可见，不同枚举中的同名成员会产生冲突。修改后：

```c
// 修改前（冲突）
enum 词元类型枚举 { 错误 = 81, ... };
enum 诊断严重级别 { 错误 = 0, ... };  // 冲突！

// 修改后（无冲突）
enum 词元类型枚举 { 词元类型枚举_错误 = 81, ... };
enum 诊断严重级别 { 诊断严重级别_诊断_错误 = 0, ... };  // 无冲突
```
