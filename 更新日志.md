#### 2. 类分析器修复 - 虚函数查找

##### 问题描述
`CN语言验收/03_函数定义/重写函数.cn` 编译时报错：
- 派生类重写基类虚函数时，编译器报"函数签名不匹配"
- 根本原因：`cn_find_virtual_member` 函数只查找当前类，不递归查找基类

##### 解决方案
修改 `src/semantics/class_analyzer.c` 中的 `cn_find_virtual_member` 函数，使其递归查找基类：

```c
CnVirtualMemberInfo *cn_find_virtual_member(CnClassInfo *class_info, const char *name) {
    // 先在当前类中查找
    for (size_t i = 0; i < class_info->virtual_member_count; i++) {
        if (strcmp(class_info->virtual_members[i].name, name) == 0) {
            return &class_info->virtual_members[i];
        }
    }
    
    // 递归查找基类
    if (class_info->base_class) {
        return cn_find_virtual_member(class_info->base_class, name);
    }
    
    return NULL;
}
```

##### 修改文件
- `src/semantics/class_analyzer.c` - 修改 `cn_find_virtual_member` 函数支持递归查找
