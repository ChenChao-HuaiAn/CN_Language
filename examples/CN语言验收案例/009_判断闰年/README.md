# 判断闰年 - CN语言验收案例

## 案例说明
这是CN语言验收案例中的第9个案例（阶段一：基础入门），演示如何根据输入的年份判断是否为闰年。

## 闰年规则
根据公历闰年规则：
1. 能被400整除的年份是闰年
2. 不能被400整除但能被100整除的年份不是闰年
3. 不能被100整除但能被4整除的年份是闰年
4. 其他情况都不是闰年

## 代码结构
- `leap_year_checker.cn` - CN语言源代码
- `run_leap_year_checker.ps1` - PowerShell运行脚本

## 运行方法
在当前目录下执行：
```powershell
.\run_leap_year_checker.ps1
```

## 验证内容
- 演示了算术运算（取模运算符%）的应用
- 展示了布尔类型和条件判断语句的使用
- 验证了函数定义与调用的机制
- 通过多个测试用例验证闰年判断的准确性

## 测试用例
程序测试了以下年份：
- 2020年：闰年（能被4整除但不能被100整除）
- 2021年：平年（不能被4整除）
- 1900年：平年（能被100整除但不能被400整除）
- 2000年：闰年（能被400整除）
- 2024年：闰年（能被4整除但不能被100整除）
