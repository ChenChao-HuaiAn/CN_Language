# 计算三角形面积 - CN语言验收案例

## 案例说明
这是CN语言验收案例中的第10个案例（阶段一：基础入门），演示如何计算三角形面积。

## 实现方法
本案例展示了两种计算三角形面积的方法：

1. **底边和高计算法**：
   - 使用公式：面积 = (底边 × 高) ÷ 2
   - 适用于已知底边和高的情况

2. **海伦公式计算法**：
   - 使用公式：面积 = √[s(s-a)(s-b)(s-c)]
   - 其中 s = (a+b+c)/2 是半周长
   - 适用于已知三边长的情况
