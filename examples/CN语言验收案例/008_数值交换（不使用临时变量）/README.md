# 数值交换（不使用临时变量）- CN语言验收案例

## 案例说明
这是CN语言验收案例中的第8个案例（阶段一：基础入门），演示如何在不使用临时变量的情况下交换两个变量的值。

## 实现方法
本案例展示了两种不使用临时变量的数值交换方法：

1. **加减法交换**：
   - 利用数学运算的性质进行交换
   - `a = a + b; b = a - b; a = a - b;`
   - 适用于大多数数值类型

2. **乘除法交换**：
   - 利用乘除法的性质进行交换
   - `a = a * b; b = a / b; a = a / b;`
   - 仅适用于非零数值，避免除零错误

- `run_swap_without_temp.ps1` - PowerShell运行脚本

## 运行方法
在当前目录下执行：

```powershell
.\run_swap_without_temp.ps1
```

- 展示了条件判断语句的使用
- 验证了数值交换的正确性

## 注意事项
- 加减法交换需注意整数溢出问题
- 乘除法交换只能用于非零数值
- 这些方法主要用于演示目的，在实际编程中仍推荐使用临时变量以提高代码可读性
