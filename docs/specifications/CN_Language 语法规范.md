# CN_Language 语法规范

## 1. 概述

CN_Language 是一个面向中文用户设计的编程语言，旨在提供直观、友好的中文编程体验。本规范定义了 CN_Language 的完整语法规则，包括词法元素、数据类型、控制结构、函数定义、模块系统等。

### 设计目标
- **中文友好**：使用中文关键字和标识符，降低语言学习门槛
- **类型安全**：支持静态类型检查和类型推断
- **现代特性**：包含结构体、枚举、模块系统等现代语言特性
- **易于扩展**：支持标准库和用户自定义模块

## 2. 字符集与编码

- **编码**：完全支持 UTF-8 编码
- **标识符字符**：
  - 中文字符（Unicode 范围：0x4E00-0x9FFF）
  - 英文字母（A-Z, a-z）
  - 数字（0-9）
  - 下划线（_）
- **空白字符**：空格、制表符、换行符、回车符
- **注释字符**：`//`（单行注释）、`/* */`（多行注释）

## 3. 词法元素

### 3.1 注释
```cn
// 单行注释

/*
  多行注释
  可以跨越多行
*/
```

### 3.2 标识符
- 必须以中文字符、英文字母或下划线开头
- 后续字符可以是中文字符、英文字母、数字或下划线
- 区分大小写
- 示例：`变量名`、`calculateSum`、`_temp`

## 4. 数据类型

### 4.1 基本类型

| 类型 | 描述 | 示例 |
|------|------|------|
| `整数` | 整数值 | `42`、`-100` |
| `小数` | 浮点数值 | `3.14`、`-0.5` |
| `字符串` | 文本字符串 | `"Hello"`、`'世界'` |
| `布尔` | 布尔值 | `真`、`假` |

### 4.2 复合类型

#### 4.2.1 数组
- 动态大小，可包含任意类型元素
- 使用方括号访问：`数组名[索引]`
- 支持长度属性：`数组名.长度()`

#### 4.2.2 结构体
- 用户自定义的复合数据类型
- 包含命名字段，每个字段有特定类型
- 支持嵌套结构体

#### 4.2.3 枚举
- 命名的常量集合
- 可指定数值或自动递增
- 支持比较操作

### 4.3 指针和引用类型

#### 4.3.1 指针类型
指针类型使用`*`符号表示，指向其他变量或值的内存地址。

```cn
// 基本指针语法
变量 a = 100;
变量* p = &a;           // 指向整数的指针
变量* q = &p;           // 指向指针的指针（二级指针）

// 类型化指针声明
整数* int_ptr = &a;     // 明确声明为整数指针
```

#### 4.3.2 引用类型
引用类型使用`&`符号表示，是变量的别名。

```cn
变量 b = 200;
整数& ref = b;          // 整数引用，ref是b的别名
```

#### 4.3.4 指针操作
```cn
// 取地址操作
//变量 地址 = &值;        // 获取值的地址
整数 a = 32;
变量 p = &a;

// 解引用操作
//值 = *指针;            // 获取指针指向的值
变量 q = *p;

// 指针算术
//指针 = 指针 + 1;       // 指针向前移动一个元素
p = p + 1;
//指针 = 指针 - 1;       // 指针向后移动一个元素
p = p - 1;
// 通过指针修改值
//*指针 = 新值;          // 修改指针指向的值
*p = 40;

```

### 4.4 类型声明语法

#### 显式类型声明
```cn
整数 年龄 = 25;
小数 价格 = 99.99;
字符串 姓名 = "张三";
布尔 已婚 = 假;
数组 数字列表 = [1, 2, 3];
整数* 年龄地址 = &年龄;     // 指针类型声明
整数& 年龄引用 = 年龄;      // 引用类型声明
```

#### 类型推断（使用 `变量` 关键字）
```cn
变量 数量 = 10;           // 推断为整数
变量 消息 = "你好世界";    // 推断为字符串
变量 标志 = 真;           // 推断为布尔
变量 列表 = [1, 2, 3];    // 推断为数组
变量* 指针 = &数量;       // 推断为指针类型
变量& 引用 = 数量;        // 推断为引用类型
```

## 6. 控制结构

### 6.1 条件语句

#### 基本形式
```cn
如果 (条件) {
    // 条件为真时执行的代码
}
```

#### 带 else 分支
```cn
如果 (条件) {
    // 条件为真时执行的代码
} 否则 {
    // 条件为假时执行的代码
}
```

#### 多重条件
```cn
如果 (条件1) {
    // 条件1为真时执行
} 否则 如果 (条件2) {
    // 条件2为真时执行
} 否则 {
    // 所有条件都为假时执行
}
```

### 6.2 循环语句

#### while 循环
```cn
当 (条件) {
    // 条件为真时重复执行
}
```

#### for 循环
```cn
循环 (初始化; 条件; 更新) {
    // 循环体
}

// 示例
循环 (变量 i = 0; i < 10; i = i + 1) {
    打印(i);
}
```

#### 循环控制
```cn
循环 (变量 i = 0; i < 10; i = i + 1) {
    如果 (i == 5) {
        中断;  // 跳出循环
    }
    
    如果 (i % 2 == 0) {
        继续;  // 跳过本次循环剩余部分
    }
    
    打印(i);
}
```

## 12. 错误处理

### 12.1 函数返回值约定
- 成功返回 `真` 或非零值
- 失败返回 `假` 或 `0`
- 错误信息通过全局变量或回调函数传递

### 12.2 示例
```cn
函数 布尔 打开文件(字符串 文件名) {
    如果 (文件不存在) {
        返回 假;
    }
    返回 真;
}
```

## 13. 语法示例

### 13.1 完整程序示例
```cn
// CN_Language 完整示例程序
模块 数学工具 {
    函数 整数 加(整数 a, 整数 b) {
        返回 a + b;
    }
    
    函数 整数 乘(整数 a, 整数 b) {
        返回 a * b;
    }
}

结构体 人员 {
    字符串 姓名;
    整数 年龄;
    小数 身高;
}

枚举 颜色 {
    红色,
    绿色,
    蓝色
}

函数 主程序() {
    导入 数学工具;
    
    // 变量声明
    变量 姓名 = "张三";
    整数 年龄 = 25;
    小数 身高 = 1.75;
    布尔 已婚 = 假;
    
    // 结构体使用
    变量 张三 = 人员{姓名, 年龄, 身高};
    打印("姓名: " + 张三.姓名);
    打印("年龄: " + 张三.年龄);
    
    // 枚举使用
    变量 主题色 = 颜色.红色;
    如果 (主题色 == 颜色.红色) {
        打印("主题色是红色");
    }
    
    // 数学计算
    变量 和 = 数学工具.加(10, 20);
    变量 积 = 数学工具.乘(10, 20);
    打印("10 + 20 = " + 和);
    打印("10 * 20 = " + 积);
    
    // 控制结构
    循环 (变量 i = 1; i <= 5; i = i + 1) {
        如果 (i % 2 == 0) {
            打印(i + " 是偶数");
        } 否则 {
            打印(i + " 是奇数");
        }
    }
    
    // 数组操作
    变量 数字列表 = [1, 2, 3, 4, 5];
    打印("数组长度: " + 数字列表.长度);
    
    返回 0;
}
```

### 13.2 标准库使用示例
```cn
函数 主程序() {
    // 数学函数
    打印("圆周率: " + 圆周率());
    打印("平方根(25): " + 平方根(25));
    
    // 字符串函数
    变量 文本 = "  Hello World!  ";
    打印("原始: '" + 文本 + "'");
    打印("修剪后: '" + 修剪(文本) + "'");
    打印("大写: '" + 转大写(文本) + "'");
    
    // 类型转换
    打印("转字符串(123): '" + 转字符串(123) + "'");
    打印("转数字('456'): " + 转数字("456"));
    
    返回 0;
}
```

## 14. 附录：语法摘要

### 14.1 BNF 语法摘要
```
程序 ::= (模块定义 | 函数定义 | 结构体定义 | 枚举定义)* 主程序函数

模块定义 ::= "模块" 标识符 "{" (函数定义 | 变量声明 | 结构体定义 | 枚举定义)* "}"

函数定义 ::= "函数" 类型? 标识符 "(" 参数列表? ")" "{" 语句* "}"

结构体定义 ::= "结构体" 标识符 "{" (类型 标识符 ";")* "}"

枚举定义 ::= "枚举" 标识符 "{" 枚举项 ("," 枚举项)* "}"
枚举项 ::= 标识符 ("=" 表达式)?

主程序函数 ::= "函数" "主程序" "(" ")" "{" 语句* "}"

语句 ::= 变量声明 ";"
       | 表达式 ";"
       | "如果" "(" 表达式 ")" "{" 语句* "}" ("否则" "如果" "(" 表达式 ")" "{" 语句* "}")* ("否则" "{" 语句* "}")?
       | "当" "(" 表达式 ")" "{" 语句* "}"
       | "循环" "(" 表达式? ";" 表达式? ";" 表达式? ")" "{" 语句* "}"
       | "中断" ";"
       | "继续" ";"
       | "返回" 表达式? ";"

变量声明 ::= ("变量" | 类型) 标识符 ("=" 表达式)?

类型 ::= "整数" | "小数" | "字符串" | "布尔" | "数组" | "列表" | 标识符
类型修饰符 ::= "*" | "&"
完整类型 ::= 类型 类型修饰符?

表达式 ::= 字面量
         | 标识符
         | 表达式 "." 标识符
         | 表达式 "[" 表达式 "]"
         | 表达式 运算符 表达式
         | "(" 表达式 ")"
         | 函数调用

函数调用 ::= 标识符 "(" 参数列表? ")"
参数列表 ::= 表达式 ("," 表达式)*

字面量 ::= 整数 | 小数 | 字符串 | "真" | "假" | "空" | 数组字面量 | 结构体字面量

数组字面量 ::= "[" (表达式 ("," 表达式)*)? "]"
结构体字面量 ::= 标识符 "{" (表达式 ("," 表达式)*)? "}"

运算符 ::= "+" | "-" | "*" | "/" | "%"
         | "==" | "!=" | "<" | ">" | "<=" | ">="
         | "与" | "且" | "或" | "非"
         | "=" | "+=" | "-=" | "*=" | "/="
```
