# CN语言 Python风格模块系统设计 TODO 列表

> 本文档定义 CN_Language 模块系统从「单文件内模块」升级为「文件即模块 + 包系统」的详细设计方案。

## 一、设计目标

将CN语言模块系统升级为类似Python的模块管理系统，实现：

1. **文件即模块**：一个 `.cn` 文件就是一个模块，文件名即模块名
2. **包系统**：包含 `__包__.cn` 的目录成为一个包
3. **跨文件导入**：支持从其他文件导入模块和成员
4. **模块搜索路径**：支持配置模块搜索路径

## 三、新增核心特性设计

### 3.1 文件模块（File Module）

一个 `.cn` 文件自动成为一个模块，文件名（不含扩展名）即模块名：

```cn
// 文件: 数学工具.cn（文件名即模块名「数学工具」）

// 公开块：以下成员对外可见
公开:
    整数 圆周率近似 = 3;

    函数 加(整数 a, 整数 b) -> 整数 {
        返回 a + b;
    }

    函数 乘(整数 a, 整数 b) -> 整数 {
        返回 a * b;
    }

// 私有块：以下成员仅文件内可见
私有:
    函数 内部辅助() -> 整数 {
        返回 0;
    }
```

**文件模块规则**：
- `公开:` 和 `私有:` 是**块结构**，声明后的所有成员都属于该可见性，直到遇到下一个可见性声明
- 未标注可见性的成员默认为 `私有`
- 同一可见性块内的成员共享相同的可见性，无需重复声明
- 文件名必须是合法的CN语言标识符（支持中文）
- 文件编码必须为 UTF-8

### 3.2 包系统（Package System）

包含 `__包__.cn` 文件的目录成为一个包：

```
项目/
├── 主程序.cn              # 入口文件
├── 工具/                   # 「工具」包
│   ├── __包__.cn          # 包初始化文件
│   ├── 数学.cn            # 「工具.数学」模块
│   ├── 字符串.cn          # 「工具.字符串」模块
│   └── 高级/              # 「工具.高级」子包
│       ├── __包__.cn      # 子包初始化文件
│       └── 矩阵.cn        # 「工具.高级.矩阵」模块
└── 配置/                   # 「配置」包
    ├── __包__.cn
    └── 默认.cn
```

**`__包__.cn` 文件作用**：
- 标识目录为包
- 可以包含包级别的初始化代码
- 可以重新导出子模块成员
- 类似 Python 的 `__init__.py`

**`__包__.cn` 示例**：
```cn
// 文件: 工具/__包__.cn

// 重新导出子模块，使用户可以直接从包导入
从 ./数学 导入 *;
从 ./字符串 导入 *;

// 包级别的公开成员
公开:
    字符串 版本 = "1.0.0";
    
    函数 获取版本() -> 字符串 {
        返回 版本;
    }
```

### 3.3 导入语法设计

#### 3.3.1 保持现有语法兼容

```cn
// 现有语法，继续支持

// 1. 全量导入（导入所有公开成员，可直接使用）
导入 数学工具;
加(1, 2);                    // 直接使用

// 2. 选择性导入（只导入指定成员）
导入 数学工具 { 加, 乘 };
加(1, 2);

// 3. 模块别名（括号语法，已实现）
导入 数学工具(数学);
数学.加(1, 2);               // 通过别名访问
```

#### 3.3.2 新增「从...导入」语法

```cn
// 新增语法：「从」关键字

// 1. 从文件模块导入指定成员
从 数学工具 导入 { 加, 圆周率近似 };
加(1, 2);

// 2. 从文件模块导入所有公开成员
从 数学工具 导入 *;
加(1, 2);

// 3. 从包导入模块
从 工具 导入 数学;
数学.加(1, 2);

// 4. 从包导入模块并设别名（使用括号语法）
从 工具 导入 数学(数);
数.加(1, 2);

// 5. 从包内模块导入成员（点分路径）
从 工具.数学 导入 { 加 };
加(1, 2);

// 6. 从子包导入
从 工具.高级.矩阵 导入 { 创建矩阵 };
```

#### 3.3.3 相对导入语法

```cn
// 相对导入（仅在包内文件中使用）

// 从同级目录导入
从 ./同级模块 导入 { 成员 };

// 从父级目录导入
从 ../父级模块 导入 { 成员 };

// 从祖父目录导入
从 ../../祖父模块 导入 { 成员 };

// 从父级的子包导入
从 ../其他包/模块 导入 { 成员 };
```

### 3.4 导入语法总结

| 语法 | 说明 | 类比 Python |
|------|------|-------------|
| `导入 模块;` | 全量导入，可直接使用成员 | `from module import *` |
| `导入 模块 { 成员 };` | 选择性导入指定成员 | `from module import name` |
| `导入 模块(别名);` | 导入模块并设别名 | `import module as alias` |
| `从 模块 导入 { 成员 };` | 从模块导入指定成员 | `from module import name` |
| `从 模块 导入 *;` | 导入所有公开成员 | `from module import *` |
| `从 包/模块 导入 { 成员 };` | 从包内模块导入（斜杠路径） | `from package.module import name` |
| `从 包.模块 导入 { 成员 };` | 从包内模块导入（点分路径） | `from package.module import name` |
| `从 ./ 导入 模块;` | 相对导入同级模块 | `from . import module` |
| `从 ../ 导入 模块;` | 相对导入父级模块 | `from .. import module` |

## 四、模块搜索路径

### 4.1 搜索顺序

模块按以下顺序搜索：

1. **当前文件所在目录**
2. **项目根目录**（包含入口文件的目录）
3. **`CN_MODULE_PATH` 环境变量**指定的路径（冒号/分号分隔）
4. **系统标准库路径**
   - Windows: `%CNLANG_HOME%\lib`
   - Linux/macOS: `/usr/lib/cnlang/lib`

### 4.2 编译器参数

```bash
# 指定额外的模块搜索路径
cnc -I 路径1 -I 路径2 源文件.cn

# 指定入口文件
cnc --main 主程序.cn

# 编译整个项目（自动发现依赖）
cnc --project 项目目录/
```

## 七、AST 节点设计

### 7.1 文件模块节点

```c
// 文件模块 AST 节点
typedef struct CnAstFileModule {
    char *file_path;              // 文件路径
    char *module_name;            // 模块名（从文件名派生）
    CnAstStmt **public_members;   // 公开成员列表
    size_t public_count;
    CnAstStmt **private_members;  // 私有成员列表
    size_t private_count;
    CnAstStmt **imports;          // 导入语句列表
    size_t import_count;
} CnAstFileModule;
```

### 7.2 扩展导入语句节点

```c
// 扩展后的导入语句 AST 节点
typedef struct CnAstImportStmt {
    CnAstImportKind kind;         // 导入类型
    char *module_path;            // 模块路径（支持点分路径）
    char *alias;                  // 别名（可选）
    char **selected_members;      // 选择性导入的成员名
    size_t member_count;
    bool is_wildcard;             // 是否为通配符导入（*）
    bool is_relative;             // 是否为相对导入
    int relative_level;           // 相对导入层级（. = 1, .. = 2）
    bool use_from_syntax;         // 是否使用「从...导入」语法
} CnAstImportStmt;

typedef enum CnAstImportKind {
    CN_IMPORT_FULL,               // 全量导入：导入 模块;
    CN_IMPORT_SELECTIVE,          // 选择性导入：导入 模块 { 成员 };
    CN_IMPORT_ALIAS,              // 别名导入：导入 模块(别名);
    CN_IMPORT_FROM_SELECTIVE,     // 从模块导入：从 模块 导入 { 成员 };
    CN_IMPORT_FROM_WILDCARD,      // 通配符导入：从 模块 导入 *;
    CN_IMPORT_FROM_MODULE,        // 从包导入模块：从 包 导入 模块;
} CnAstImportKind;
```

```cn

// 在同一文件内定义内联模块
模块 本地工具 {
    公开:
        函数 辅助() -> 整数 {
            返回 加(1, 2);   // 使用导入的函数

        函数 计算(整数 x) -> 整数 {
            返回 x * 2;
        }

    变量 结果 = 本地工具.辅助();
    返回 0;
}
```
