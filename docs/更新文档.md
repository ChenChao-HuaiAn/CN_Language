### 📝 CN语言构造函数/析构函数语法

根据 [`plans/011 CN语言面向对象编程支持方案.md`](plans/011 CN语言面向对象编程支持方案.md)：

| 功能 | 语法设计 | 说明 |
|------|---------|------|
| 构造函数 | `函数 类名()` | 与类同名的函数 |
| 析构函数 | `函数 ~类名()` | 波浪号前缀 |

**示例代码**：
```cn
类 点 {
公开:
    整数 x;
    整数 y;
    函数 点() {        // 构造函数
        x = 0;
        y = 0;
    }
    函数 ~点() {       // 析构函数
        // 清理资源
    }
}
```
