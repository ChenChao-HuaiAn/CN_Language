### CN语言标准库全中文函数名实现

#### 概述
为CN语言标准库的84个纯英文函数名添加了中文别名，实现"双轨制"命名体系，使CN语言程序可以使用纯中文函数名进行开发。

#### 使用示例

```c
// 使用中文函数名
初始化运行时();
打印整数(42);
打印换行();

void* ptr = 分配内存(100);
设置内存(ptr, 0, 100);
释放内存(ptr);

CnMutex mutex;
初始化互斥锁(&mutex);
加互斥锁(&mutex);
解互斥锁(&mutex);
销毁互斥锁(&mutex);

退出运行时();
```

#### 技术实现
- 使用 `#define` 宏定义实现中文别名
- 默认启用中文别名，可通过定义 `CN_NO_CHINESE_NAMES` 禁用
- 完全向后兼容，英文函数名仍然可用
- 无运行时开销，宏定义在编译时展开
