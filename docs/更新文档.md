1. **添加 `cn_sem_is_same_symbol()` 辅助函数**（第18-63行）
   - 判断两个符号是否来自同一个模块的同一个成员
   - 判断依据：符号种类、名字、声明作用域（`decl_scope`）或类型指针
   - 对于没有 `decl_scope` 的符号，回退到比较类型指针或函数参数数量

   - 检测到同名符号时，先调用 `cn_sem_is_same_symbol()` 判断是否是同一符号
   - 如果是同一符号，静默忽略，不报错也不重复添加
   - 如果是不同符号冲突，才报错

3. **修改选择性导入逻辑**（两处）
   - 同上，使用相同的去重逻辑

4. **修复 `导入 xxx;` 语法的错误行为**
   - 添加了 `!import->use_from_syntax` 检查
   - 确保 `导入 xxx;` 语法只注册模块符号，不会错误地全量导入模块成员
