### 五、剩余问题分析

#### 5.1 🟠 中优先级：参数类型不匹配 (arg_kind=1, param_kind=8)

**错误特征**：
```
语义错误：函数 '创建IR指令' 参数 0 类型不匹配 (arg_kind=1, param_kind=8)
```

**影响范围**：
- IR生成器模块：IR生成.cn
- 诊断模块：诊断恢复.cn、诊断消息表.cn、诊断修复.cn

**根本原因**：
`arg_kind=1` 表示整数类型，`param_kind=8` 表示指针类型。函数期望指针参数，但传入了整数。

**修复建议**：
检查函数调用时是否需要取地址操作符 `&`。

---

#### 5.2 🟡 低优先级：case值必须是常量表达式

**错误特征**：
```
语义错误：case 值必须是常量表达式
```

**影响范围**：
- IR生成器模块：IR生成.cn

**根本原因**：
switch语句的case值使用了非常量表达式。

**修复建议**：
确保case值使用常量或枚举值。

---

#### 5.3 🟡 低优先级：模块导入错误

**错误特征**：
```
模块导入后符号未正确解析
```

**影响范围**：
- 模块加载器：加载器.cn

**根本原因**：
模块导入语句处理可能存在问题。
