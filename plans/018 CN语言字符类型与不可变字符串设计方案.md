# 018 CN语言字符类型与不可变字符串设计方案

## 1. 概述

### 1.1 背景

当前CN语言存在以下问题：
- 缺少 `字符` 类型，无法声明字符数组
- `字符串` 类型是 `char*` 指针，不是字符数组
- 扫描器等底层代码无法正确编译

### 1.2 设计目标

1. 添加 `字符` 类型支持底层字节操作
2. 将 `字符串` 改造为不可变字符串类型
3. 保持与C语言的兼容性
4. 向后兼容现有代码

---

## 2. 类型系统设计

### 2.1 字符类型

| 类型 | 含义 | C语言对应 | 用途 |
|------|------|----------|------|
| `字符` | 单个字符/字节 | `char` | 底层字节操作 |
| `字符*` | 字符指针 | `char*` | 字符串指针 |
| `字符[]` | 字符数组 | `char[]` | 缓冲区、字符串操作 |

### 2.2 不可变字符串类型

| 类型 | 含义 | C语言对应 | 用途 |
|------|------|----------|------|
| `字符串` | 不可变字符串 | `const char*` | 高层文本处理 |

## 3. 语法设计

### 3.1 字符类型语法

#### 3.1.1 字符变量声明

```cn
// 单个字符
字符 ch = 'A';
字符 换行 = '\n';

// 字符指针
字符* ptr = "Hello";

// 字符数组
字符 缓冲区[256];
字符 名字[32] = "CN语言";
```

#### 3.1.2 字符数组初始化

```cn
// 字符数组初始化
字符 缓冲区[256] = {0};
字符 消息[] = "Hello, World!";

// 从字符数组创建字符串
字符 数据[64];
字符串 result = 数据;  // 隐式转换为不可变字符串
```

### 3.2 不可变字符串语法

#### 3.2.1 字符串声明

```cn
// 字符串字面量
字符串 name = "CN语言";
字符串 greeting = "你好，世界！";

// 字符串是不可变的，不能修改
// name[0] = 'c';  // 编译错误！
```

#### 3.2.2 字符串操作

```cn
// 字符串连接
字符串 full = 连接("Hello", " World");

// 字符串长度
整数 len = 字符串长度(name);

// 字符串比较
如果 (比较字符串(a, b) == 0) {
    打印("相等");
}

// 字符串查找
整数 pos = 查找字符串(text, "CN");
```

### 3.3 类型转换

#### 3.3.1 隐式转换

```cn
字符 缓冲区[64] = "Hello";
字符串 s = 缓冲区;  // 字符[] -> 字符串（安全）

字符* ptr = 缓冲区;  // 字符[] -> 字符*（兼容C）
字符串 s2 = ptr;     // 字符* -> 字符串（安全）
```

#### 3.3.2 显式转换

```cn
字符串 s = "Hello";

// 获取底层字符指针（只读）
字符* ptr = 获取字符指针(s);

// 复制到字符数组
字符 缓冲区[64];
复制字符串(缓冲区, s);
```

## 4. 代码生成

### 4.1 字符类型代码生成

| CN代码 | 生成的C代码 |
|--------|------------|
| `字符 ch = 'A';` | `char ch = 'A';` |
| `字符* ptr = "Hello";` | `char* ptr = "Hello";` |
| `字符 缓冲区[256];` | `char buffer[256];` |
| `字符 名字[] = "CN";` | `char name[] = "CN";` |

### 4.2 字符串类型代码生成

| CN代码 | 生成的C代码 |
|--------|------------|
| `字符串 s = "Hello";` | `const char* s = "Hello";` |
| `字符串 连接(a, b);` | `cn_string_concat(a, b);` |
| `字符串长度(s)` | `cn_string_length(s);` |

### 4.3 运行时函数

```c
// 字符串操作函数（运行时库）
const char* cn_string_concat(const char* a, const char* b);
int cn_string_length(const char* s);
int cn_string_compare(const char* a, const char* b);
const char* cn_string_find(const char* s, const char* sub);
char* cn_string_copy(char* dest, const char* src);
```

## 5. 语义检查

### 5.1 类型兼容性规则

| 源类型 | 目标类型 | 允许 | 说明 |
|--------|---------|------|------|
| `字符` | `整数` | ✅ | 隐式转换 |
| `整数` | `字符` | ✅ | 截断低8位 |
| `字符[]` | `字符*` | ✅ | 数组退化 |
| `字符[]` | `字符串` | ✅ | 安全转换 |
| `字符*` | `字符串` | ✅ | 安全转换 |
| `字符串` | `字符*` | ⚠️ | 需要显式转换 |
| `字符串` | `字符[]` | ❌ | 禁止（不可变） |

### 5.2 不可变性检查

```cn
字符串 s = "Hello";

// ❌ 编译错误：字符串是不可变的
s[0] = 'h';

// ❌ 编译错误：字符串是不可变的
s = 修改(s);  // 必须重新赋值

// ✅ 正确：创建新字符串
字符串 s2 = 连接(s, " World");
```

## 6. 向后兼容性

### 6.1 现有代码兼容

现有使用 `字符串` 的代码无需修改：

```cn
// 现有代码
字符串 name = "CN语言";
打印(name);
打印字符串(name);
```

生成的C代码从 `char*` 变为 `const char*`，语义更安全。

### 6.2 迁移指南

需要修改字符数组相关代码：

```cn
// 旧代码（不工作）
整数 缓冲区[256];  // 错误：int buffer[256]

// 新代码（正确）
字符 缓冲区[256];  // 正确：char buffer[256]
```

## 7. 实现计划

### 7.1 阶段一：字符类型支持

1. **词法分析器**：添加 `字符` 关键字
2. **语法分析器**：支持 `字符`、`字符*`、`字符[]` 类型声明
3. **语义检查**：字符类型兼容性检查
4. **代码生成**：生成正确的C类型

### 7.2 阶段二：不可变字符串

1. **类型系统**：`字符串` 类型改为 `const char*`
2. **语义检查**：不可变性检查
3. **运行时库**：字符串操作函数

### 7.3 阶段三：字符串操作增强

1. **内置函数**：`连接`、`字符串长度`、`比较字符串` 等
2. **类型转换**：安全的类型转换函数
3. **文档更新**：语法规范和编码规范（“plans\001 CN Language语法规范设计文档.md”、“.ai-coder\skills\cn-coding-standards\SKILL.md”）
