## 1. 概述

### 1.1 目标

扩展CN语言标准库，新增文件操作高级功能和三种核心数据结构（动态数组、链表、哈希表），提升CN语言的实用性和开发效率。

### 1.2 意义

1. **完善文件操作**：现有文件操作功能较为基础，需要扩展以支持更复杂的文件处理场景
2. **提供数据结构支持**：动态数组、链表、哈希表是程序开发中最常用的数据结构
3. **保持语言一致性**：所有新增函数均提供中文别名，与CN语言设计理念一致
4. **内存安全**：利用现有的内存管理基础设施，确保内存安全

## 3. 数据结构扩展设计

### 3.1 动态数组（Vector）

#### 3.1.1 数据结构定义

```c
/**
 * 动态数组结构
 */
typedef struct {
    void* data;           // 数据存储区
    size_t size;          // 当前元素数量
    size_t capacity;      // 当前容量
    size_t elem_size;     // 每个元素的大小
} CnVector;
```

#### 3.1.2 核心操作函数

| 函数 | 中文别名 | 功能描述 |
|------|---------|---------|
| `cn_rt_vector_create(size_t elem_size, size_t init_capacity)` | `创建动态数组` | 创建动态数组 |
| `cn_rt_vector_destroy(CnVector* vec)` | `销毁动态数组` | 销毁动态数组 |
| `cn_rt_vector_push(CnVector* vec, const void* elem)` | `尾部添加` | 在尾部添加元素 |
| `cn_rt_vector_pop(CnVector* vec, void* out_elem)` | `尾部弹出` | 弹出尾部元素 |
| `cn_rt_vector_get(const CnVector* vec, size_t index, void* out_elem)` | `获取元素` | 获取指定位置元素 |
| `cn_rt_vector_set(CnVector* vec, size_t index, const void* elem)` | `设置元素` | 设置指定位置元素 |
| `cn_rt_vector_size(const CnVector* vec)` | `动态数组长度` | 获取元素数量 |
| `cn_rt_vector_capacity(const CnVector* vec)` | `动态数组容量` | 获取当前容量 |
| `cn_rt_vector_clear(CnVector* vec)` | `清空动态数组` | 清空所有元素 |
| `cn_rt_vector_resize(CnVector* vec, size_t new_capacity)` | `调整容量` | 调整数组容量 |

#### 3.1.3 实现思路

```c
// 创建动态数组
CnVector* cn_rt_vector_create(size_t elem_size, size_t init_capacity) {
    CnVector* vec = (CnVector*)cn_rt_malloc(sizeof(CnVector));
    if (!vec) return NULL;
    
    vec->elem_size = elem_size;
    vec->capacity = init_capacity > 0 ? init_capacity : 8;
    vec->size = 0;
    vec->data = cn_rt_calloc(vec->capacity, elem_size);
    
    if (!vec->data) {
        cn_rt_free(vec);
        return NULL;
    }
    return vec;
}

// 尾部添加元素（自动扩容）
int cn_rt_vector_push(CnVector* vec, const void* elem) {
    if (vec->size >= vec->capacity) {
        // 扩容策略：2倍增长
        size_t new_capacity = vec->capacity * 2;
        void* new_data = cn_rt_realloc(vec->data, new_capacity * vec->elem_size);
        if (!new_data) return -1;
        vec->data = new_data;
        vec->capacity = new_capacity;
    }
    memcpy((char*)vec->data + vec->size * vec->elem_size, elem, vec->elem_size);
    vec->size++;
    return 0;
}
```

#### 3.1.4 内存管理策略

- **初始容量**：默认8个元素，可自定义
- **扩容策略**：2倍扩容，均摊O(1)时间复杂度
- **缩容策略**：暂不实现，避免频繁内存操作
- **内存释放**：销毁时释放所有内存

### 3.2 链表（LinkedList）

#### 3.2.1 数据结构定义

```c
/**
 * 链表节点
 */
typedef struct CnListNode {
    void* data;                    // 数据指针
    struct CnListNode* prev;       // 前驱节点
    struct CnListNode* next;       // 后继节点
} CnListNode;

/**
 * 双向链表结构
 */
typedef struct {
    CnListNode* head;              // 头节点
    CnListNode* tail;              // 尾节点
    size_t size;                   // 元素数量
    size_t elem_size;              // 每个元素的大小
} CnLinkedList;
```

#### 3.2.2 核心操作函数

| 函数 | 中文别名 | 功能描述 |
|------|---------|---------|
| `cn_rt_list_create(size_t elem_size)` | `创建链表` | 创建链表 |
| `cn_rt_list_destroy(CnLinkedList* list)` | `销毁链表` | 销毁链表 |
| `cn_rt_list_push_front(CnLinkedList* list, const void* elem)` | `头部插入` | 在头部插入元素 |
| `cn_rt_list_push_back(CnLinkedList* list, const void* elem)` | `尾部插入` | 在尾部插入元素 |
| `cn_rt_list_pop_front(CnLinkedList* list, void* out_elem)` | `头部删除` | 删除头部元素 |
| `cn_rt_list_pop_back(CnLinkedList* list, void* out_elem)` | `尾部删除` | 删除尾部元素 |
| `cn_rt_list_size(const CnLinkedList* list)` | `链表长度` | 获取元素数量 |
| `cn_rt_list_clear(CnLinkedList* list)` | `清空链表` | 清空所有元素 |
| `cn_rt_list_get_front(const CnLinkedList* list, void* out_elem)` | `获取头部` | 获取头部元素 |
| `cn_rt_list_get_back(const CnLinkedList* list, void* out_elem)` | `获取尾部` | 获取尾部元素 |

#### 3.2.3 实现思路

```c
// 创建链表
CnLinkedList* cn_rt_list_create(size_t elem_size) {
    CnLinkedList* list = (CnLinkedList*)cn_rt_malloc(sizeof(CnLinkedList));
    if (!list) return NULL;
    
    list->head = NULL;
    list->tail = NULL;
    list->size = 0;
    list->elem_size = elem_size;
    return list;
}

// 头部插入
int cn_rt_list_push_front(CnLinkedList* list, const void* elem) {
    CnListNode* node = (CnListNode*)cn_rt_malloc(sizeof(CnListNode));
    if (!node) return -1;
    
    node->data = cn_rt_malloc(list->elem_size);
    if (!node->data) {
        cn_rt_free(node);
        return -1;
    }
    memcpy(node->data, elem, list->elem_size);
    
    node->prev = NULL;
    node->next = list->head;
    
    if (list->head) {
        list->head->prev = node;
    } else {
        list->tail = node;  // 空链表
    }
    list->head = node;
    list->size++;
    return 0;
}
```

#### 3.2.4 内存管理策略

- **节点分配**：每个节点独立分配，支持任意位置插入删除
- **数据存储**：每个节点存储数据副本，避免外部数据生命周期问题
- **内存释放**：销毁时遍历释放所有节点和数据

### 3.3 哈希表（HashMap）

#### 3.3.1 数据结构定义

```c
/**
 * 哈希表节点
 */
typedef struct CnMapNode {
    char* key;                      // 键（字符串）
    void* value;                    // 值
    struct CnMapNode* next;         // 链地址法解决冲突
} CnMapNode;

/**
 * 哈希表结构
 */
typedef struct {
    CnMapNode** buckets;            // 桶数组
    size_t bucket_count;            // 桶数量
    size_t size;                    // 元素数量
    size_t value_size;              // 值的大小
} CnHashMap;
```

#### 3.3.2 核心操作函数

| 函数 | 中文别名 | 功能描述 |
|------|---------|---------|
| `cn_rt_map_create(size_t value_size, size_t init_buckets)` | `创建哈希表` | 创建哈希表 |
| `cn_rt_map_destroy(CnHashMap* map)` | `销毁哈希表` | 销毁哈希表 |
| `cn_rt_map_insert(CnHashMap* map, const char* key, const void* value)` | `插入键值` | 插入键值对 |
| `cn_rt_map_get(const CnHashMap* map, const char* key, void* out_value)` | `获取值` | 获取键对应的值 |
| `cn_rt_map_remove(CnHashMap* map, const char* key)` | `删除键值` | 删除键值对 |
| `cn_rt_map_contains(const CnHashMap* map, const char* key)` | `包含键` | 检查键是否存在 |
| `cn_rt_map_size(const CnHashMap* map)` | `哈希表大小` | 获取元素数量 |
| `cn_rt_map_clear(CnHashMap* map)` | `清空哈希表` | 清空所有元素 |

#### 3.3.3 实现思路

```c
// 哈希函数（djb2算法）
static unsigned long cn_rt_hash(const char* str) {
    unsigned long hash = 5381;
    int c;
    while ((c = *str++)) {
        hash = ((hash << 5) + hash) + c;  // hash * 33 + c
    }
    return hash;
}

// 创建哈希表
CnHashMap* cn_rt_map_create(size_t value_size, size_t init_buckets) {
    CnHashMap* map = (CnHashMap*)cn_rt_malloc(sizeof(CnHashMap));
    if (!map) return NULL;
    
    map->bucket_count = init_buckets > 0 ? init_buckets : 16;
    map->size = 0;
    map->value_size = value_size;
    map->buckets = (CnMapNode**)cn_rt_calloc(map->bucket_count, sizeof(CnMapNode*));
    
    if (!map->buckets) {
        cn_rt_free(map);
        return NULL;
    }
    return map;
}

// 插入键值对
int cn_rt_map_insert(CnHashMap* map, const char* key, const void* value) {
    unsigned long index = cn_rt_hash(key) % map->bucket_count;
    
    // 检查是否已存在
    CnMapNode* node = map->buckets[index];
    while (node) {
        if (strcmp(node->key, key) == 0) {
            memcpy(node->value, value, map->value_size);  // 更新值
            return 0;
        }
        node = node->next;
    }
    
    // 创建新节点
    CnMapNode* new_node = (CnMapNode*)cn_rt_malloc(sizeof(CnMapNode));
    if (!new_node) return -1;
    
    new_node->key = cn_rt_string_dup(key);
    new_node->value = cn_rt_malloc(map->value_size);
    if (!new_node->key || !new_node->value) {
        cn_rt_free(new_node->key);
        cn_rt_free(new_node->value);
        cn_rt_free(new_node);
        return -1;
    }
    memcpy(new_node->value, value, map->value_size);
    
    // 头插法
    new_node->next = map->buckets[index];
    map->buckets[index] = new_node;
    map->size++;
    return 0;
}
```

#### 3.3.4 内存管理策略

- **桶数量**：默认16个桶，可自定义
- **冲突解决**：链地址法，简单高效
- **负载因子**：暂不实现自动扩容，保持简单
- **键类型**：仅支持字符串键，简化实现
- **内存释放**：销毁时释放所有节点、键和值
