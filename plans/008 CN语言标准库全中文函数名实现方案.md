## 1. 概述

### 1.1 目标

为CN语言标准库的所有纯英文函数名添加中文别名，实现"双轨制"命名体系，使CN语言程序可以使用纯中文函数名进行开发，提升代码可读性和中文用户体验。

### 1.2 意义

1. **语言一致性**：CN语言使用中文关键字，标准库函数也应支持中文命名
2. **降低学习门槛**：中文函数名更符合中文用户的思维习惯
3. **代码可读性**：纯中文代码更易于理解和维护
4. **向后兼容**：保留英文函数名，不影响现有代码

### 1.3 现状分析

| 分类 | 数量 | 占比 |
|------|------|------|
| 已有中文别名（双轨制） | 52个 | 35% |
| 纯中文函数名 | 14个 | 9% |
| 纯英文函数名（需添加中文别名） | 84个 | 56% |

## 2. 命名规范

### 2.1 中文函数名命名规则

1. **动词+名词结构**：如 `分配内存`、`释放内存`
2. **简洁明了**：避免冗长，如 `求幂` 而非 `计算幂运算结果`
3. **语义准确**：中文含义与英文函数功能完全对应
4. **一致性**：同类函数使用相似命名模式

### 2.2 命名模式对照表

| 英文前缀 | 中文前缀 | 示例 |
|----------|----------|------|
| `init` | 初始化 | `cn_rt_interrupt_init` → `初始化中断` |
| `destroy` | 销毁 | `cn_rt_mutex_destroy` → `销毁互斥锁` |
| `enable` | 启用 | `cn_rt_interrupt_enable` → `启用中断` |
| `disable` | 禁用 | `cn_rt_interrupt_disable` → `禁用中断` |
| `register` | 注册 | `cn_rt_interrupt_register` → `注册中断` |
| `unregister` | 注销 | `cn_rt_interrupt_unregister` → `注销中断` |
| `lock` | 加锁 | `cn_rt_mutex_lock` → `加互斥锁` |
| `unlock` | 解锁 | `cn_rt_mutex_unlock` → `解互斥锁` |
| `load` | 读取 | `cn_rt_atomic32_load` → `原子读取32` |
| `store` | 写入 | `cn_rt_atomic32_store` → `原子写入32` |
| `alloc` | 分配 | `cn_rt_array_alloc` → `分配数组` |
| `free` | 释放 | `cn_rt_array_free` → `释放数组` |
| `print` | 打印 | `cn_rt_print_int` → `打印整数` |
| `read` | 读取 | `cn_rt_mem_read` → `读取内存` |
| `write` | 写入 | `cn_rt_mem_write` → `写入内存` |

### 2.3 特殊命名约定

| 场景 | 命名规则 | 示例 |
|------|----------|------|
| 原子操作 | `原子` + 操作 + 位宽 | `原子加法32`、`原子交换64` |
| 锁操作 | 锁类型 + 操作 | `加互斥锁`、`加自旋锁`、`读锁定` |
| 安全操作 | `安全` + 操作 | `安全读取内存`、`安全写入内存` |
| 类型转换 | 源类型 + `转` + 目标类型 | `整数转字符串`、`小数转字符串` |

## 3. 实现方案

### 3.1 技术方案：宏定义别名

使用C语言宏定义实现中文函数名别名，这是最简单且兼容性最好的方案：

```c
// 在头文件中添加宏定义
#define 初始化运行时 cn_rt_init
#define 退出运行时 cn_rt_exit
```

**优点**：
- 实现简单，无需修改现有实现代码
- 编译时展开，无运行时开销
- 完全向后兼容
- 易于维护

**缺点**：
- 调试时显示的是英文函数名
- IDE自动补全可能需要额外配置

### 3.2 实现位置

在现有头文件末尾添加中文别名宏定义区块：

```c
// =============================================================================
// 中文函数名别名 (Chinese Function Name Aliases)
// =============================================================================
#ifdef CN_USE_CHINESE_NAMES

// 运行时生命周期管理
#define 初始化运行时 cn_rt_init
#define 退出运行时 cn_rt_exit

// ... 更多别名定义

#endif // CN_USE_CHINESE_NAMES
```

### 3.3 条件编译控制

提供编译选项控制是否启用中文别名：

```c
// 默认启用中文别名
#ifndef CN_NO_CHINESE_NAMES
#define CN_USE_CHINESE_NAMES
#endif
```

    void* ptr = 分配内存(100);
