## 1. 任务概述

### 1.1 任务目标

将现有C语言实现的IR生成器（`src/ir/`）使用CN语言重写，实现自托管编译器的IR生成功能。

### 1.2 功能要求

| 功能项 | 说明 | 优先级 |
|--------|------|--------|
| IR指令定义 | 定义IR指令集 | ⭐⭐⭐ 高 |
| IR模块生成 | 从AST生成IR模块 | ⭐⭐⭐ 高 |
| IR函数生成 | 生成函数IR | ⭐⭐⭐ 高 |
| IR基本块生成 | 生成基本块 | ⭐⭐⭐ 高 |
| IR指令生成 | 生成各类指令 | ⭐⭐⭐ 高 |
| 优化Pass | 常量折叠、死代码消除等 | ⭐⭐ 中 |

### 1.3 验收标准

- [ ] IR生成器能正确生成所有IR结构
- [ ] IR指令集完整覆盖所有操作
- [ ] 优化Pass正常工作
- [ ] IR生成测试用例100%通过
- [ ] 与C版本IR生成器输出一致

## 3. 详细任务描述

### 3.1 IR指令定义

```cn
// src_cn/IR生成器/IR指令.cn
模块 IR生成器 {
公开:
    // IR指令操作码
    枚举 IR操作码 {
        // 算术运算
        加法,
        减法,
        乘法,
        除法,
        取模,
        
        // 比较运算
        等于,
        不等于,
        小于,
        大于,
        小于等于,
        大于等于,
        
        // 逻辑运算
        与,
        或,
        非,
        
        // 内存操作
        加载,
        存储,
        分配,
        栈分配,
        
        // 控制流
        跳转,
        条件跳转,
        调用,
        返回,
        
        // 函数操作
        参数,
        获取参数,
        
        // 类型转换
        转换,
        截断,
        扩展,
        
        // 其他
        标签,
        函数定义,
        函数声明
    }
    
    // IR值类型
    枚举 IR值类型 {
        整数常量,
        浮点常量,
        字符串常量,
        寄存器,
        参数,
        函数,
        基本块
    }
    
    // IR指令结构
    类 IR指令 {
    私有:
        IR操作码 操作码;
        IR值 目标;
        IR值 操作数1;
        IR值 操作数2;
        类型* 结果类型;
        源位置 位置;
        
    公开:
        函数 创建(IR操作码 操作码) -> IR指令*;
        函数 设置目标(IR值 目标值) -> 空类型;
        函数 设置操作数1(IR值 值) -> 空类型;
        函数 设置操作数2(IR值 值) -> 空类型;
    }
}
```

### 3.5 优化Pass设计

```cn
// src_cn/IR生成器/优化Pass.cn
模块 IR生成器 {
公开:
    // 优化Pass基类
    类 优化Pass {
    公开:
        虚拟 函数 执行(IR模块* 模块) -> 布尔;
        虚拟 函数 获取名称() -> 字符串;
    }
    
    // 常量折叠Pass
    类 常量折叠Pass : 优化Pass {
    公开:
        函数 执行(IR模块* 模块) -> 布尔 重写 {
            变量 改变 = 假;
            循环 变量 函数 在 模块.获取函数列表() {
                改变 = 改变 或 折叠函数(函数);
            }
            返回 改变;
        }
        
    私有:
        函数 折叠函数(IR函数* 函数) -> 布尔;
        函数 折叠指令(IR指令* 指令) -> 布尔;
    }
    
    // 死代码消除Pass
    类 死代码消除Pass : 优化Pass {
    公开:
        函数 执行(IR模块* 模块) -> 布尔 重写 {
            变量 改变 = 假;
            循环 变量 函数 在 模块.获取函数列表() {
                改变 = 改变 或 消除死代码(函数);
            }
            返回 改变;
        }
        
    私有:
        函数 消除死代码(IR函数* 函数) -> 布尔;
        函数 计算活跃变量(IR函数* 函数) -> 集合;
    }
}
```
