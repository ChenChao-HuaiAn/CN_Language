## 3. 详细任务描述

### 3.1 符号表设计

```cn
// src_cn/语义分析器/符号表.cn
模块 语义分析器 {
公开:
    // 符号类型
    枚举 符号类型 {
        变量符号,
        函数符号,
        类型符号,
        模块符号,
        参数符号,
        成员符号
    }
    
    // 符号结构
    类 符号 {
    私有:
        字符串 名称;
        符号类型 类型;
        类型* 符号类型值;
        源位置 定义位置;
        布尔 是否常量;
        布尔 是否静态;
        布尔 是否公开;
        
    公开:
        函数 创建(字符串 名称, 符号类型 类型) -> 符号*;
        函数 获取名称() -> 字符串 { 返回 名称; }
        函数 获取类型() -> 符号类型 { 返回 类型; }
        函数 是常量() -> 布尔 { 返回 是否常量; }
    }
    
    // 作用域结构
    类 作用域 {
    私有:
        作用域* 父作用域;
        映射 符号表;
        作用域类型 类型;
        
    公开:
        函数 创建(作用域* 父, 作用域类型 类型) -> 作用域*;
        函数 声明符号(符号* 符号值) -> 布尔;
        函数 查找符号(字符串 名称) -> 符号*;
        函数 查找本地符号(字符串 名称) -> 符号*;
    }
}
```

### 3.2 类型系统设计

```cn
// src_cn/语义分析器/类型系统.cn
模块 语义分析器 {
公开:
    // 类型类别
    枚举 类型类别 {
        基础类型,      // 整数、小数、布尔、字符串
        数组类型,      // 数组
        结构体类型,    // 结构体
        枚举类型,      // 枚举
        函数类型,      // 函数
        类类型,        // 类
        接口类型,      // 接口
        指针类型,      // 指针
        空类型,        // void
        类型参数       // 泛型参数
    }
    
    // 类型结构
    类 类型 {
    私有:
        类型类别 类别;
        字符串 名称;
        整数 大小;
        布尔 是否可变;
        
    公开:
        函数 创建基础类型(字符串 名称, 整数 大小) -> 类型*;
        函数 创建数组类型(类型* 元素类型, 整数 大小) -> 类型*;
        函数 创建函数类型(类型* 返回类型, 类型列表 参数) -> 类型*;
        函数 创建指针类型(类型* 目标类型) -> 类型*;
        
        函数 是否相同(类型* 其他) -> 布尔;
        函数 是否可隐式转换为(类型* 目标) -> 布尔;
        函数 是否可显式转换为(类型* 目标) -> 布尔;
    }
    
    // 基础类型常量
    常量 类型* 整数类型 = 类型.创建基础类型("整数", 4);
    常量 类型* 小数类型 = 类型.创建基础类型("小数", 8);
    常量 类型* 布尔类型 = 类型.创建基础类型("布尔", 1);
    常量 类型* 字符串类型 = 类型.创建基础类型("字符串", 8);
    常量 类型* 空类型值 = 类型.创建基础类型("空类型", 0);
}
```

### 3.3 语义分析器类设计

```cn
// src_cn/语义分析器/语义分析器.cn
模块 语义分析器 {
公开:
    类 语义分析器 {
    私有:
        作用域* 当前作用域;
        符号表* 全局符号表;
        类型系统* 类型系统实例;
        诊断管理器* 诊断;
        整数 错误计数;
        
    公开:
        函数 创建(诊断管理器* 诊断) -> 语义分析器*;
        函数 销毁(语义分析器* 实例) -> 空类型;
        函数 分析程序(程序节点* 程序) -> 布尔;
        
    私有:
        // 第一遍：收集符号
        函数 收集符号(程序节点* 程序) -> 空类型;
        函数 收集函数声明(函数声明* 声明) -> 空类型;
        函数 收集变量声明(变量声明* 声明) -> 空类型;
        函数 收集类声明(类声明* 声明) -> 空类型;
        
        // 第二遍：类型检查
        函数 类型检查(程序节点* 程序) -> 空类型;
        函数 检查函数声明(函数声明* 声明) -> 空类型;
        函数 检查语句块(语句块* 块) -> 空类型;
        函数 检查语句(语句节点* 语句) -> 空类型;
        函数 检查表达式(表达式节点* 表达式) -> 类型*;
        
        // 作用域管理
        函数 进入作用域(作用域类型 类型) -> 空类型;
        函数 退出作用域() -> 空类型;
        函数 声明符号(符号* 符号值) -> 布尔;
        函数 查找符号(字符串 名称) -> 符号*;
        
        // 类型检查辅助
        函数 检查赋值类型(表达式节点* 目标, 表达式节点* 值) -> 布尔;
        函数 检查函数调用(函数调用* 调用) -> 布尔;
        函数 检查成员访问(成员访问* 访问) -> 类型*;
        函数 检查二元运算(二元表达式* 表达式) -> 类型*;
    }
}
```
