## 1. 任务概述

### 1.1 任务目标

为CN语言自托管编译器提供运行时库的CN语言接口，使编译器能够使用运行时库的功能。

### 1.2 功能要求

| 功能项 | 说明 | 优先级 |
|--------|------|--------|
| 内存管理接口 | 内存分配、释放、重分配 | ⭐⭐⭐ 高 |
| 字符串操作接口 | 字符串连接、比较、长度等 | ⭐⭐⭐ 高 |
| 动态数组接口 | 向量操作 | ⭐⭐⭐ 高 |
| 哈希表接口 | 键值对存储 | ⭐⭐⭐ 高 |
| 文件操作接口 | 文件读写 | ⭐⭐⭐ 高 |
| 命令行参数接口 | 参数获取和解析 | ⭐⭐⭐ 高 |
| 类型信息接口 | 运行时类型信息 | ⭐⭐ 中 |

### 1.3 验收标准

- [ ] 所有运行时接口在CN中可用

### 3.1 内存管理接口

```cn
// src_cn/运行时/内存.cn
模块 运行时 {
公开:
    // 内存分配
    函数 分配内存(整数 大小) -> 指针 {
        返回 cn_rt_malloc(大小);
    }
    
    // 内存释放
    函数 释放内存(指针 内存) -> 空类型 {
        cn_rt_free(内存);
    }
    
    // 内存重分配
    函数 重新分配(指针 内存, 整数 新大小) -> 指针 {
        返回 cn_rt_realloc(内存, 新大小);
    }
    
    // 内存复制
    函数 复制内存(指针 目标, 指针 源, 整数 大小) -> 空类型 {
        cn_rt_memcpy(目标, 源, 大小);
    }
    
    // 内存设置
    函数 设置内存(指针 目标, 整数 值, 整数 大小) -> 空类型 {
        cn_rt_memset(目标, 值, 大小);
    }
    
    // 内存比较
    函数 比较内存(指针 左, 指针 右, 整数 大小) -> 整数 {
        返回 cn_rt_memcmp(左, 右, 大小);
    }
}
```

```cn
// src_cn/运行时/命令行.cn
模块 运行时 {

    }

    函数 参数个数() -> 整数 {
        返回 cn_rt_cli_argc();
    }
    
    // 获取指定参数
    函数 获取参数(整数 索引) -> 字符串 {
        返回 cn_rt_cli_argv(索引);
    }
    
    // 获取程序名称
    函数 程序名称() -> 字符串 {
        返回 cn_rt_cli_program_name();
    }
    
    // 查找选项
    函数 查找选项(字符串 选项名) -> 字符串 {
        返回 cn_rt_cli_find_option(选项名);
    }
    
    // 检查选项是否存在
    函数 选项存在(字符串 选项名) -> 布尔 {
        返回 cn_rt_cli_has_option(选项名);
    }
    
    // 获取所有参数
    函数 所有参数() -> 字符串[] {
        变量 个数 = 参数个数();
        变量 结果 = 新建 字符串[个数];
        循环 变量 i = 0; i < 个数; i = i + 1 {
            结果[i] = 获取参数(i);
        }
        返回 结果;
    }
}
```
