## 3. 实现方案说明

### 3.1 关键发现

根据 [`plans/001 CN Language语法规范设计文档.md`](plans/001 CN Language语法规范设计文档.md) 分析：

1. **CN语言没有"外部函数声明"语法** - 不能使用 `外部函数` 声明C函数
2. **CN语言不支持 `类型定义` 语法** - 不能使用 `类型 xxx = YYY;` 创建类型别名
3. **运行时库已提供中文接口** - 在 `include/cnlang/runtime/stdlib.h` 中已定义

### 3.2 实现策略

**方案：直接使用运行时库提供的中文接口**

CN语言运行时库（`include/cnlang/runtime/stdlib.h`）已提供完整的中文接口，CN代码可直接调用：

```cn
// 直接调用运行时库提供的中文接口
函数 主程序() -> 整数 {
    整数 参数数量 = 获取参数个数();
    字符串 参数值 = 获取参数(0);
    打印("参数: " + 参数值);
    返回 0;
}
```

### 3.3 需要扩展的接口

运行时库中尚未提供的接口，需要在C层面扩展：

| 接口类别 | 已有接口 | 需要扩展的接口 |
|---------|---------|---------------|
| 动态数组 | 无 | `创建数组()`、`销毁数组()`、`数组添加()` 等 |
| 哈希表 | 无 | `创建哈希表()`、`哈希表插入()`、`哈希表获取()` 等 |

## 4. 详细任务描述

### 4.1 第一阶段：扩展C运行时库

在 `include/cnlang/runtime/stdlib.h` 中添加动态数组和哈希表的中文接口。

**注意**：C头文件中的中文接口必须使用 **C语言类型**（如 `void*`、`size_t`），而不是CN语言的中文类型名。

```c
// 动态数组接口（使用C语言类型）
CnVector* 创建数组(size_t 初始容量);
void 销毁数组(CnVector* 数组);
int 数组添加(CnVector* 数组, void* 元素);
void* 数组获取(CnVector* 数组, size_t 索引);
size_t 数组长度(CnVector* 数组);
void 清空数组(CnVector* 数组);

// 哈希表接口（使用C语言类型）
CnMap* 创建哈希表(size_t 初始容量);
void 销毁哈希表(CnMap* 表);
int 哈希表插入(CnMap* 表, const char* 键, void* 值);
void* 哈希表获取(CnMap* 表, const char* 键);
int 哈希表包含(CnMap* 表, const char* 键);
size_t 哈希表大小(CnMap* 表);
void 清空哈希表(CnMap* 表);
```
