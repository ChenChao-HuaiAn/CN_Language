# CHANGELOG

## [1.0.0] - 2026-01-29

### 🎉 CN_Language 1.0.0 正式发布！

这是 CN_Language 编程语言的第一个正式版本，标志着中文编程语言进入生产就绪阶段。

### 核心特性

#### 编译器 (cnc)
- **完整的前端实现**：词法分析、语法分析、AST 构建
- **语义分析**：类型检查、名称解析、作用域管理
- **中间表示 (IR)**：三地址码生成与优化
- **C 代码后端**：生成高效的 C 代码输出

#### 语言特性
- **基础类型**：整数、小数、布尔、字符串
- **复合类型**：数组、结构体、枚举
- **指针支持**：包括函数指针
- **控制流**：如果/否则、当、循环、选择
- **函数**：支持参数、返回值、递归调用
- **模块系统**：模块声明、导入、可见性控制（公开/私有）
- **预处理器**：宏定义、条件编译、文件包含

#### 工具链
- **cnfmt**：代码格式化工具
- **cncheck**：静态代码检查工具
- **cnlsp**：语言服务器协议支持
- **cnrepl**：交互式解释器
- **cnperf**：性能分析工具

#### 运行时库 (cn_runtime)
- **内存管理**：Arena 分配器、安全内存操作
- **I/O 支持**：控制台输入输出、文件操作
- **数学库**：基础数学函数
- **字符串库**：字符串操作函数
- **同步原语**：锁、条件变量

#### 操作系统开发支持
- **Freestanding 模式**：无标准库依赖
- **内存映射**：底层内存操作
- **内联汇编**：汇编语法支持

### 测试覆盖
- 106 个测试用例全部通过
- 覆盖单元测试、集成测试、系统测试
- 多平台验证（Windows/Linux/macOS）

### 已知限制
- "从...导入"语法的符号直接导入功能尚未完全实现，请使用点号访问（如 `模块.成员`）
- 部分预留关键字（类、接口、模板等）已定义但功能未实现

### 文档
- 完整的语言规范文档
- 代码风格规范（C 和 CN）
- 测试规范和发布流程规范
